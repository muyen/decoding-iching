# 卦之時義分析：時義如何修正爻位吉凶

*研究日期：2026-01-20*

---

## 摘要

本研究探討易經六十四卦的「時義」（核心主題）如何與爻位理論交互作用，產生吉凶判斷的修正效果。通過系統性分析雜卦傳、彖傳的卦德描述，以及384條爻辭的吉凶統計，本研究發現：

1. **時義係數**可將卦分為四大類：順時卦、逆時卦、轉換卦、等待卦
2. **位置吉凶可被時義逆轉**：三爻在「謙」卦反吉，五爻在「明夷」卦反凶
3. **周易卦序反映時義演進**：上經從創生到穩定，下經從感應到未濟

基於雜卦傳的核心概念與彖傳的時義描述，我將64卦分為四大類：

## 第二部分：時義修正表

### 2.1 「逆轉」案例分析

#### 2.1.1 本凶變吉（位凶 + 順時 = 吉）

根據爻位理論，三爻為「內卦極點，多凶多懼」，基礎值為 -1。然而以下案例顯示時義可以逆轉此凶象：

| 卦名 | 爻辭 | 位置預期 | 實際判斷 | 時義修正原因 |
|------|------|----------|----------|--------------|
| **謙卦九三** | 勞謙，君子有終，**吉** | 三爻凶 | **吉** | 謙之時義：「天道益謙」，越謙越吉。九三「勞謙」恰好體現謙德極致 |
| **復卦六三** | 頻復，厲**无咎** | 三爻凶 | **無咎** | 復之時義：「剛反」，頻繁回復正道雖危但終無咎 |
| **晉卦六三** | 眾允，**悔亡** | 三爻凶 | **悔亡** | 晉之時義：「柔進而上行」，眾人允許則悔恨消亡 |
| **升卦九三** | 升虛邑 | 三爻凶 | **中性** | 升之時義：「柔以時升」，上升至空虛之邑無阻礙 |
| **益卦六三** | 益之用凶事，**无咎** | 三爻凶 | **無咎** | 益之時義：「損上益下」，即使用於凶事也能化解 |

**發現規律 A：三爻在「謙、復、晉、升、益」等順時卦中，因時義助力而化凶為吉。**

#### 2.1.2 本吉變凶（位吉 + 逆時 = 凶）

根據爻位理論，五爻為「九五之尊，最佳位」，基礎值為 +2。然而以下案例顯示：

| 卦名 | 爻辭 | 位置預期 | 實際判斷 | 時義修正原因 |
|------|------|----------|----------|--------------|
| **明夷六五** | 箕子之明夷，利**艱貞** | 五爻吉 | **需艱守** | 明夷時義：「明入地中」，光明受傷之時，即使尊位也需韜光養晦 |
| **困卦九五** | 劓刖，困于赤紱 | 五爻吉 | **初凶後解** | 困之時義：「剛揜」，困窮之時，五爻雖尊也受困 |
| **蹇卦九五** | 大蹇，朋來 | 五爻吉 | **需等待助力** | 蹇之時義：「險在前」，困難之時，五爻也需朋友相助 |
| **恆卦六五** | 恆其德，婦人吉，**夫子凶** | 五爻吉 | **男子凶** | 恆之時義：「久于其道」，柔居五位須柔順，剛則凶 |
| **歸妹六五** | 其君之袂不如娣良 | 五爻吉 | **不如下位** | 歸妹時義：「位不當」，嫁妹之時，正位反不如偏位美好 |

**發現規律 B：五爻在「明夷、困、蹇、恆、歸妹」等逆時卦中，因時義壓制而吉中帶凶。**

#### 2.1.3 上爻逆轉案例

上爻為「物極必反」，基礎值為 -1，然而：

| 卦名 | 爻辭 | 位置預期 | 實際判斷 | 時義修正原因 |
|------|------|----------|----------|--------------|
| **謙卦上六** | 鳴謙，利用行師，征邑國 | 上爻凶 | **可征伐** | 謙之時義使上位也得利 |
| **大有上九** | 自天祐之，**吉无不利** | 上爻凶 | **大吉** | 大有時義：「柔得尊位」，功成名就之卦，上爻得天佑 |
| **否卦上九** | 傾否，先否**後喜** | 上爻凶 | **轉吉** | 否之時義走到極點反轉，否極泰來 |
| **剝卦上九** | 碩果不食，君子**得輿** | 上爻凶 | **君子吉** | 剝之時義：唯一陽爻在上，碩果僅存 |

**發現規律 C：上爻在否極泰來的轉折點（否、剝）或極盛之卦（大有、謙）中可變吉。**

### 2.3 時義修正公式（進階版）

基於以上案例分析，我提出進階的時義修正公式：

```
最終爻義 = (位置基礎值 + 得位 + 得中 + 相應 + 承乘) × 時義係數 × 卦序係數

時義係數：
- 順時卦：+1.5（大力助吉）
- 等待卦：+1.0（維持原狀）
- 轉換卦：±1.0（視時機而定）
- 逆時卦：-0.5 至 -1.5（壓制或反轉）

卦序係數：
- 上經卦（1-30）：穩定係數 1.0
- 下經卦（31-64）：變動係數 1.1
```

## 第三部分：卦序演進分析

### 3.1 周易卦序的時義邏輯

周易六十四卦的排列（King Wen Sequence）並非隨機，而是反映了時義的有機演進：

#### 3.1.1 上經三十卦：從創生到穩定

```
創始階段（1-4卦）：乾→坤→屯→蒙
├── 乾坤：天地開闢，萬物資始資生
├── 屯：難生，天造草昧
└── 蒙：啟蒙，養正聖功

發展階段（5-12卦）：需→訟→師→比→小畜→履→泰→否
├── 需訟：等待與爭端
├── 師比：戰爭與親附
├── 小畜履：積蓄與踐行
└── 泰否：通泰與閉塞（首次大循環）

成熟階段（13-20卦）：同人→大有→謙→豫→隨→蠱→臨→觀
├── 同人大有：團結與豐收
├── 謙豫：謙德與歡樂
├── 隨蠱：隨順與整治
└── 臨觀：臨眾與觀示

轉折階段（21-30卦）：噬嗑→賁→剝→復→无妄→大畜→頤→大過→坎→離
├── 噬嗑賁：刑法與文飾
├── 剝復：剝落與回復（第二次大循環）
├── 无妄大畜：無妄與積蓄
├── 頤大過：頤養與過極
└── 坎離：水火定位（上經終結）
```

**上經時義演進主題：天地創生 → 社會發展 → 成熟治理 → 物極必反**

```

├── 夬姤：決斷與邂逅

├── 革鼎：去故取新
├── 震艮：起動與止息
```

### 4.1 時義吉凶預測公式

```

其中：
- 卦德類型 = {順時:+1.5, 等待:+1.0, 轉換:±1.0, 逆時:-1.0}

- 卦序位置 = 上經(1.0) 或 下經(1.1)

最終判斷 = Σ(所有因子)
若 > 2: 大吉
若 1~2: 吉/無咎

```

```
三爻吉凶 = IF(卦德∈{謙,復,晉,升,益,渙,解})
           THEN 可化凶為吉
           ELSE 維持本凶

```

#### 公式三：五爻折損條件

```
五爻吉凶 = IF(卦德∈{明夷,困,蹇,恆,歸妹,否,剝})

           ELSE 維持本吉

```

## 第五部分：新發現

### 5.1 發現一：謙卦高吉率的時義機制

謙卦是吉率最高的卦之一（83.3%，5吉0凶）。通過本研究發現其機制：

```
謙卦時義係數 = +2.0（超級順時）

彖傳四重加持：
1. 天道：「天道虧盈而益謙」
2. 地道：「地道變盈而流謙」
3. 鬼神：「鬼神害盈而福謙」
4. 人道：「人道惡盈而好謙」

結果：大幅提升吉率
（注：沒有任何卦是100%六爻皆吉）
```

**啟示：當時義足夠強大時，可完全覆蓋位置的負面因素。**

### 5.2 發現二：「剝復循環」的位置逆轉

剝卦與復卦形成一個完整的循環：

```
剝卦：陽消陰長（從下往上剝）
├── 初六：剝床以足（凶）
├── 六二：剝床以辨（凶）
├── 六三：剝之（無咎！）← 逆轉點
├── 六四：剝床以膚（凶）
├── 六五：貫魚（無不利）
└── 上九：碩果不食（君子吉）← 物極必反

復卦：陽長陰消（從下往上復）
├── 初九：不遠復（元吉！）← 起點最吉
├── 六二：休復（吉）
├── 六三：頻復（厲無咎）← 轉折
├── 六四：中行獨復（中性）
├── 六五：敦復（無悔）
└── 上六：迷復（凶）← 終點反凶
```

**發現規律：在消長型卦象中，初爻和上爻的吉凶完全相反：**
- 消退卦（剝）：初凶上吉
- 回復卦（復）：初吉上凶

### 5.3 發現三：「二五相應」的時義放大效應

當二爻與五爻形成陰陽相應時，時義的放大效應最強：

| 卦名 | 二爻 | 五爻 | 相應性質 | 時義效果 |
|------|------|------|----------|----------|
| 屯 | 六二 | 九五 | 陰陽應 | 「屯其膏」受限 |
| 蒙 | 九二 | 六五 | 陽陰應 | 「包蒙吉」極吉 |
| 比 | 六二 | 九五 | 陰陽應 | 「比之自內」吉 |
| 泰 | 九二 | 六五 | 陽陰應 | 「帝乙歸妹」元吉 |
| 否 | 六二 | 九五 | 陰陽應 | 「休否」大人吉 |
| 謙 | 六二 | 六五 | 同陰無應 | 各自吉，無加成 |

**發現規律：二五相應 + 順時卦 = 吉凶效果放大；二五相應 + 逆時卦 = 吉凶效果轉化。**

### 5.4 發現四：「時義大矣哉」的特殊卦群

彖傳中有九卦被明確標示「時義大矣哉」或「時用大矣哉」：

### 5.5 發現五：卦序的「反復」結構

## 結論

本研究通過系統性分析，揭示了「卦之時義」對爻位吉凶的修正機制：

1. **時義可逆轉爻位吉凶**：三爻本凶，在謙、復、升等順時卦中反吉；五爻本吉，在明夷、困等逆時卦中反需隱忍。

2. **卦序反映時義演進**：上經從天地創生到水火定位，下經從男女感應到既未濟循環。

3. **可預測的規律**：
   - 時義係數 × 位置基礎值 = 最終吉凶
   - 二五相應可放大時義效果
   - 「時義大矣哉」九卦需特別審時度勢

4. **新公式**：TMF（時義修正因子）= f(卦德, 爻位, 卦序) 可用於預測爻辭吉凶

這些發現為易經研究提供了可量化、可預測的分析框架，將傳統的定性解讀推進到定量分析的層次。
