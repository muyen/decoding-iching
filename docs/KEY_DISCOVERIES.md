## 一、顛覆性發現

### 1.1 「守正必吉」是迷思

**傳統觀點**：貞（守正）必然導致吉祥

**數據證據**：
```
貞吉：42次
貞凶：8次（16%的組合反而凶！）
貞厲：8次
貞吝：5次
```

**新見解**：易經反對教條主義。「守正」需要配合時機，在錯誤的時機固守反而招凶。

**典型案例**：
- 恆卦初六「浚恆，貞凶」— 過度追求恆久反而凶
- 節卦上六「苦節，貞凶」— 過度節制反而凶

---

### 1.2 四爻是「補救之位」而非「凶位」

**傳統觀點**：「四多懼」，四爻是危險位置

**數據證據**：
```
「无咎」在各爻位的分佈：
初爻：17.3%
二爻：14.8%
三爻：14.8%
四爻：22.2% ← 最高！
五爻：13.6%
上爻：17.3%
```

**新見解**：四爻不是天然凶位，而是**最能通過謹慎行為化險為夷的位置**。「近君之位」的風險，恰好可以通過謹慎補救。

---

### 1.3 當位影響被嚴重高估

**傳統觀點**：「當位」（陽居陽位、陰居陰位）是吉凶判斷的關鍵

**數據證據**：
```
影響吉凶的因素權重：
時義：40%
爻位：25%
得中：15%
應：10%
承乘：7%
當位：3% ← 最低！
```

**最有力證據**：
- 既濟卦：六爻**全部當位**，卻「初吉終亂」
- 未濟卦：六爻**全不當位**，卻可「亨」

**新見解**：當位只是必要條件，絕非充分條件。時義才是決定性因素。

---

### 1.4 「利見大人」只屬於領導層

**傳統觀點**：「利見大人」適用於所有爻位

**數據證據**：
```
「利見大人」出現位置：
初爻：0次
二爻：0次
三爻：0次
四爻：0次
五爻：4次 ← 幾乎全在這裡
卦辭：9次
```

**新見解**：只有處於領導地位（五爻）或從整體格局出發（卦辭），才能「見大人」。基層求見貴人是不現實的。

## 二、結構性發現

### 2.1 互卦是最強影響因素

**發現**：互卦（取2-3-4爻為下，3-4-5爻為上）對本卦意義的影響力達到 **0.7315**，遠超基線（0.1930）。

**含義**：這驗證了2000年來的「互卦」概念，不是迷信而是有統計基礎的。

**應用**：
- 屯卦互卦為剝：創業中已預示失敗風險
- 泰卦互卦為歸妹：通泰有嫁娶之喜（六五直接說「帝乙歸妹」）

---

### 2.2 King Wen配對的深意

**發現**：周文王卦序中相鄰的配對（如屯-蒙、需-訟）具有 **0.7565** 的語義相似度。

**含義**：這些配對不是隨機排列，而是有意設計的「互補視角」。

**規律**：
- 綜卦（上下顛倒）描述同一主題的不同階段
- 錯卦（陰陽全反）描述互補而非對立的概念

---

### 2.3 八個對稱卦的樞紐地位

**發現**：8個自對稱卦（乾、坤、頤、大過、坎、離、中孚、小過）旋轉180度後與自身相同。

**特殊性**：
- 錯卦即綜卦（或自身）
- 爻辭呈現內部對稱
- 是理解其他卦的基準點

**新見解**：這8卦可能代表易經的「八個基本範式」。

## 三、模式發現

### 3.1 數字象徵系統

| 數字 | 出現次數 | 象徵意義 |
|------|----------|----------|
| **三** | 45次（最多） | 完整週期（三年＝困難週期） |
| **七** | 6次 | 週期性回歸（七日來復） |
| **十** | 8次 | 極長時間/幾乎不可能（十年不克） |

**規律**：
- 「三年」一律表示一個完整困難週期
- 「七日」表示「等待後自然恢復」
- 「十年」是「幾乎不可能」的隱喻

## 六、3D立方體模型與五行發現（2026-01-21更新）

### 6.1 3×3×3 立方體模型

**概念**：將64卦×6爻 = 384個爻映射到27個「細胞」：
- X軸：下卦性質（陰/中/陽）
- Y軸：上卦性質（陰/中/陽）
- Z軸：爻位層次（下層1-2 / 中層3-4 / 上層5-6）

**Z軸主效應（最強）**：
| 層次 | 吉率 | 解釋 |
|------|------|------|
| 下層（初、二爻）| **45.3%** | 起始階段，穩健 |
| 中層（三、四爻）| **31.2%** | 變化核心，最危險 |
| 上層（五、上爻）| **39.1%** | 完成階段，適中 |

**關鍵發現**：中層（三、四爻）是整體最凶險的區域，與傳統「三多凶、四多懼」一致。

### 6.2 三才效應

| 三才位置 | 爻位 | 吉率 | 凶率 |
|----------|------|------|------|
| 人位 | 二、五爻 | 43.8% | **11.7%** ← 最低凶率 |
| 地位 | 初、四爻 | 39.1% | 17.2% |
| 天位 | 三、上爻 | 32.8% | **21.9%** ← 最高凶率 |

**新見解**：「人位」（二、五爻）是最安全的位置，而「天位」（三、上爻）最危險。這解釋了為何九二和六五傳統上被視為最佳。

### 6.3 五行關係的顛覆性發現

**傳統觀點 vs 數據實證**：

| 五行關係 | 傳統預期 | 實證吉率 | 實證凶率 | 結論 |
|----------|----------|----------|----------|------|
| **相生** | 最吉 | 44-47% | **5.6%** | ✓ 最安全 |
| **相剋** | 較凶 | 35-38% | 17% | 中等 |
| **比和** | 和諧吉 | 29.8% | **21.4%** | ✗ **最危險！** |

**顛覆性結論**：
- 「比和」（同卦/同五行）不是和諧，而是**最危險的組合**
- 「相生」才是真正安全的關係（凶率僅5.6%）
- 這挑戰了「和諧最好」的傳統觀點
