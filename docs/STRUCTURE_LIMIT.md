# 純結構預測的極限

## 研究結論

### ⚠️ 重要更正（2026-01-21）

**早期版本（V17等）聲稱達到62%準確率，但這是過擬合（overfitting）結果。**

| 版本 | 報告準確率 | 樣本量 | 問題 |
|------|------------|--------|------|
| V17 | 62.2% | 90（僅15卦） | **過擬合** |
| V18 (全驗證) | 45.1% | 384爻 | ✓ 有效 |
| ML 5-fold CV | ~50% | 384爻 | ✓ 有效 |

**真正的結構預測極限是 ~50%，不是 60%。**

### 完整384爻驗證準確率

| 版本 | 方法 | 準確率 | 驗證方式 |
|------|------|--------|----------|
| 基準 | 全猜「中」 | 46.4% | - |
| 位置規則 | 爻位預測 | 45-49% | 全集測試 |
| 結構特徵 | 位置+XOR+八卦 | ~49% | 5-fold CV |
| +五行 | 加入五行關係 | 50.3% | 5-fold CV |
| **極限** | 純結構理論極限 | **~50%** | 5-fold CV |

### 結構能解釋的規律

**100%規則（強規則）：**
1. 五爻從不凶
2. 謙卦(艮/坤)前四爻全吉
3. 觀卦(坤/巽)全中
4. 得中+無應 → 從不凶

**強傾向規則：**
1. 艮在下卦 → 58%吉率
2. 調和（有艮）→ 50%吉率
3. 止/順組合 → 67%吉率
4. 六爻 → 47%凶率

### 結構無法解釋的案例

**隨卦(017)二爻：**
```
結構特徵：
- 二爻（得中）
- 陰爻居偶位（得位）
- 與五爻相應（有應）
- 下卦震、上卦兌

預期：應該吉（結構優良）
實際：凶

原因：必須看爻辭才知道
```

這說明：**同樣的結構可以有不同結果**，取決於卦義。

### 為什麼結構有極限？

1. **卦義大於結構**
   - 謙卦為何全吉？因為「謙」的德性
   - 困卦為何多凶？因為「困」的境遇
   - 結構只是載體，卦義才是內容

2. **爻辭的行為指引**
   - 「勿用」= 結構險但不動則無過
   - 「利貞」= 需要條件才吉
   - 這些是語義，不是結構

3. **特殊象徵**
   - 「龍」在乾卦的特殊意義
   - 「馬」在坤卦的特殊意義
   - 這些無法從二進制推導

### 結構的真正價值

結構決定**基調**，語義決定**細節**：

```
結構作用 ≈ 50-60%
- 位置傾向（五爻安全，六爻危險）
- 八卦屬性（艮止為謙）
- 關係網絡（應、承、乘）

語義作用 ≈ 40-50%
- 卦義（謙、困、隨...）
- 爻辭具體指引
- 特殊象徵
```

### 五行分析的額外發現

| 五行關係 | 吉率 | 凶率 | 結論 |
|----------|------|------|------|
| 相生 | 44-47% | **5.6%** | 最安全 |
| 相剋 | 35-38% | 17% | 中等 |
| 比和 | 29.8% | **21.4%** | 最危險 |

**關鍵發現**：「比和」（同卦/同五行）反而最危險，而非傳統認為的「和諧」。

### 最終答案

**純結構預測的理論極限約為 ~50%**（經完整384爻、5-fold CV驗證）

要達到更高準確率，必須引入語義信息：
- 卦名/卦義的結構化表示
- 爻辭關鍵詞的提取
- 或者：直接使用卦號作為特徵（但這就是查表/記憶）

### 我們嘗試過的方法

1. **1D分析**：爻位（初至上）、陰陽
2. **2D分析**：8×8棋盤（上卦×下卦）、XOR值
3. **3D分析**：3×3×3立方體（下卦性質×上卦性質×層次）
4. **關係分析**：承乘比應、得位得中
5. **卦序分析**：文王序、馬王堆序、Gray碼
6. **八宮分析**：傳統八宮歸屬
7. **五行分析**：上下卦五行生剋
8. **機器學習**：Naive Bayes、Logistic Regression、KNN、Perceptron

**結論**：無論用什麼方法組合，結構預測極限都在 ~50% 左右。
