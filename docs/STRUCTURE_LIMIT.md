# 純結構預測的極限

## 研究結論

### 準確率進程

| 版本 | 方法 | 準確率 |
|------|------|--------|
| 隨機 | 全猜「中」 | 52.2% |
| V5 | 基礎結構 | 50.0% |
| V7 | +五行交互 | 54.4% |
| V9 | +八卦屬性+關係 | 56.7% |
| **極限** | 純結構理論極限 | ~60% |
| V8 | 卦號查表 | 100%（作弊）|

### 結構能解釋的規律

**100%規則（強規則）：**
1. 五爻從不凶
2. 謙卦(艮/坤)前四爻全吉
3. 觀卦(坤/巽)全中
4. 得中+無應 → 從不凶

**強傾向規則：**
1. 艮在下卦 → 58%吉率
2. 調和（有艮）→ 50%吉率
3. 止/順組合 → 67%吉率
4. 六爻 → 47%凶率

### 結構無法解釋的案例

**隨卦(017)二爻：**
```
結構特徵：
- 二爻（得中）
- 陰爻居偶位（得位）
- 與五爻相應（有應）
- 下卦震、上卦兌

預期：應該吉（結構優良）
實際：凶

原因：必須看爻辭才知道
```

這說明：**同樣的結構可以有不同結果**，取決於卦義。

### 為什麼結構有極限？

1. **卦義大於結構**
   - 謙卦為何全吉？因為「謙」的德性
   - 困卦為何多凶？因為「困」的境遇
   - 結構只是載體，卦義才是內容

2. **爻辭的行為指引**
   - 「勿用」= 結構險但不動則無過
   - 「利貞」= 需要條件才吉
   - 這些是語義，不是結構

3. **特殊象徵**
   - 「龍」在乾卦的特殊意義
   - 「馬」在坤卦的特殊意義
   - 這些無法從二進制推導

### 結構的真正價值

結構決定**基調**，語義決定**細節**：

```
結構作用 ≈ 50-60%
- 位置傾向（五爻安全，六爻危險）
- 八卦屬性（艮止為謙）
- 關係網絡（應、承、乘）

語義作用 ≈ 40-50%
- 卦義（謙、困、隨...）
- 爻辭具體指引
- 特殊象徵
```

### 最終答案

**純結構預測的理論極限約為 55-60%**

要達到更高準確率，必須引入語義信息：
- 卦名/卦義的結構化表示
- 爻辭關鍵詞的提取
- 或者：直接使用卦號作為特徵（但這就是查表/記憶）
