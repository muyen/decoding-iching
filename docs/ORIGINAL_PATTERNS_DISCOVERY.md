# 易經原始規律發現

**研究日期**：2026-01-20
**方法論**：只使用原始元素（二進制結構、King Wen卦序），排除後加系統（五行、納甲）

---

## 核心問題

五行、納甲、八宮等系統是**漢代後加**的，用這些來「驗證」易經可能是循環論證。

我們需要區分：

| 原始元素（周代） | 後加元素（漢代+） |
|------------------|-------------------|
| 陰陽二進制 | 五行對應 |
| 八卦結構（2³） | 納甲天干地支 |
| 六十四卦（2⁶） | 八宮系統 |
| 爻辭文本 | 六親系統 |
| King Wen 卦序 | 方位對應 |
| 綜卦/錯卦結構 | 卦氣曆法 |

---

## 發現一：陽爻數量的 U 型曲線

### 數據

| 陽爻數 | 吉凶分 | 卦數 | 解讀 |
|--------|--------|------|------|
| 0（純陰） | **2.00** | 1 | 極吉（坤） |
| 1 | 0.83 | 6 | 偏吉 |
| 2 | 0.47 | 15 | 中性 |
| 3（平衡） | 0.45 | 20 | 中性 |
| 4 | 0.47 | 15 | 中性 |
| 5 | 0.67 | 6 | 偏吉 |
| 6（純陽） | **2.00** | 1 | 極吉（乾） |

### 規律

```
      吉
       │  ▲                           ▲
       │  █                           █
       │  █  ▄                     ▄  █
       │  █  █  ▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄  █  █
       │  █  █  █              █  █  █
      凶└──█──█──█──────────────█──█──█──
          0  1  2    3    4    5  6
              陽爻數量
```

### 洞見

**純粹比混合更穩定**

傳統「物極必反」觀點認為極端必凶，但數據顯示：
- 純陽（乾）和純陰（坤）最吉
- 陰陽混合（3陽3陰）反而最不確定

**解釋**：純粹的結構沒有內在矛盾，混合結構存在張力。

## 發現二：自對稱卦偏吉

### 數據

| 類型 | 卦數 | 吉凶分 |
|------|------|--------|
| 自對稱卦 | 8 | **0.88** |
| 非自對稱卦 | 56 | 0.52 |

### 自對稱卦列表

| 卦 | 二進制 | 時義 |
|----|--------|------|
| 乾 | 111111 | 順時 ◎ |
| 坤 | 000000 | 順時 ◎ |
| 坎 | 010010 | 逆時 ✗ |
| 離 | 101101 | 等待 ○ |
| 頤 | 100001 | 等待 ○ |
| 大過 | 011110 | 逆時 ✗ |
| 中孚 | 110011 | 順時 ◎ |
| 小過 | 001100 | 等待 ○ |

### 洞見

**結構對稱 = 內在平衡 = 穩定**

自對稱卦旋轉180°後與自身相同，這種數學性質與吉凶相關：
- 3/8 順時（37.5%）vs 全體 25%
- 只有 2/8 逆時（25%）vs 全體 22%

這是**純數學結構**的吉凶規律，不需要任何後加解釋系統。

## 發現三：King Wen 卦序的「W 型」分佈

### 數據

| 卦序位置 | 吉凶分 |
|----------|--------|
| 1-16（開頭） | **0.88** |
| 17-32 | 0.25 |
| 33-48 | 0.25 |
| 49-64（結尾） | **0.88** |

### 規律

```
吉 │  ████                              ████
   │  ████                              ████
   │  ████                              ████
中 │  ████████████████████████████████████
   │       ████████████████████████████
凶 │       ████████████████████████████
   └──────────────────────────────────────
       1-16    17-32    33-48    49-64
```

### 洞見

**King Wen 的「敘事結構」**

卦序不是隨機排列，而是有意設計：
1. **開頭（1-16）**：建立秩序（乾坤開篇）
2. **中間（17-48）**：經歷混亂和挑戰
3. **結尾（49-64）**：恢復秩序（既濟未濟收尾）

這是一個**完整的生命週期敘事**。

## 發現四：King Wen 配對的「互補設計」

### 數據

- 相鄰卦對時義一致：8/32 = **25%**
- 相鄰卦對時義不同：24/32 = **75%**

### 時義相反的配對

| 配對 | 卦名 | 時義 |
|------|------|------|
| 7-8 | 師-比 | 逆時 ↔ 順時 |
| 11-12 | 泰-否 | 順時 ↔ 逆時 |
| 23-24 | 剝-復 | 逆時 ↔ 順時 |
| 35-36 | 晉-明夷 | 順時 ↔ 逆時 |
| 53-54 | 漸-歸妹 | 順時 ↔ 逆時 |

### 洞見

**辯證思維的編碼**

King Wen 故意把互補的卦放在一起：
- 泰（通泰）→ 否（閉塞）
- 剝（剝落）→ 復（復甦）
- 晉（進升）→ 明夷（光明受傷）

這是**陰陽轉化**的結構性表達，不是巧合。

## 發現五：方位角度差（原始 vs 後加的邊界案例）

### 說明

方位對應可能是後加的（《說卦傳》），但如果存在相關性，有兩種可能：
1. 漢代人發現了真實規律
2. 漢代人根據吉凶反推方位（循環論證）

### 數據

| 方位角度差 | 吉凶分 | 解讀 |
|------------|--------|------|
| 同位（0°） | **0.88** | 最吉 |
| 鄰位（45°） | 0.69 | 偏吉 |
| 斜對（135°） | 0.69 | 偏吉 |
| 側位（90°） | 0.31 | 中性 |
| 對沖（180°） | **0.25** | 最凶 |

### 謹慎解讀

這個發現需要謹慎對待：
- **如果**方位是根據卦義推導的 → 循環論證
- **如果**方位是獨立系統 → 可能發現了真實關聯

目前無法確定是哪種情況。

## 結論：什麼是可靠的？

### 高可靠性（純原始元素）

| 發現 | 可靠性 | 原因 |
|------|--------|------|
| 陽爻數量 U 型曲線 | ★★★★★ | 純二進制分析 |
| 自對稱卦偏吉 | ★★★★★ | 純數學結構 |
| King Wen 卦序 W 型 | ★★★★★ | 原始卦序 |
| 相鄰卦對互補設計 | ★★★★★ | 原始卦序 |

### 中等可靠性（可能有後加影響）

| 發現 | 可靠性 | 原因 |
|------|--------|------|
| 方位角度差規律 | ★★★☆☆ | 方位可能是後加 |
| 互卦影響力 | ★★★☆☆ | 互卦概念可能是後加 |

### 低可靠性（後加系統）

| 發現 | 可靠性 | 原因 |
|------|--------|------|

| 納甲六親 | ★☆☆☆☆ | 納甲是漢代創立 |

## 可操作的統一公式

基於**高可靠性**發現，我們可以建立一個純原始的公式：

```
卦義基礎分 = f(陽爻數量) × 對稱性係數

其中：
f(0) = f(6) = 2.0  # 純陰純陽最吉
f(1) = f(5) = 0.75
f(2) = f(4) = 0.47
f(3) = 0.45

對稱性係數：
自對稱卦 = 1.2
非自對稱卦 = 1.0

卦序位置修正：
1-16位 = +0.3
17-48位 = 0
49-64位 = +0.3
```

這個公式只使用**無可爭議的原始元素**，可靠性最高。
