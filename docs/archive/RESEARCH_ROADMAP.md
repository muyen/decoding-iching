## 最新發現（2026-01-21 能量流動分析）

### 驗證結果

| 假設 | 結果 | 效應大小 |
|------|------|----------|
| 陽氣上升=吉 | ⚠ 效果不明顯 | 吉率差異 ~3% |
| 連續陰爻=阻塞=凶 | ✓ 部分驗證 | 5個連續陰爻凶率最高(33%) |
| **有邊界轉換=吉** | ✓ **強驗證** | **吉率39% vs 30%** |
| 適中變化=吉 | ✓ 驗證 | 3次變化最佳(38%吉) |
| 同降比同升好 | ✓ 驗證 | 凶率4% vs 29% |

### 關鍵發現

1. **邊界轉換效應**：上下卦交界處有陰陽轉換時，吉率顯著提高（+9%）
   - 這解釋了為什麼「泰」「否」的結構這麼重要

2. **同向流動悖論**：「同升」反而凶（29%凶率），「同降」反而吉（4%凶率）
   - 可能解釋：同升=競爭，同降=和諧

3. **變化次數最優值**：陰陽轉換3次時吉率最高（38.3%）
   - 0次太平緩（停滯）
   - 5次太劇烈（不穩定）

| 規則 | 傳統說法 | 驗證結果 | 效應大小 |
|------|----------|----------|----------|
| 三多凶 | 三爻位多危險 | ✓ 強驗證 | 凶率32.8%，最高 |
| 五多功 | 五爻位多功績 | ✓ 強驗證 | 吉率48.4%，最高 |
| 得中為貴 | 二五爻較吉 | ✓ 強驗證 | +18.4%吉率差異 |
| 二多譽 | 二爻位多榮譽 | ~ 部分驗證 | 吉率45.3%，次高 |
| 四多懼 | 四爻位多恐懼 | ~ 部分驗證 | 中等水平 |
| 有應者吉 | 相應爻較吉 | ~ 部分驗證 | +9.9%吉率差異 |
| 當位吉 | 陽居陽位吉 | ✗ 不成立 | <0.5%差異 |
| 陽乘陰順 | 剛統柔最佳 | ✗ 不成立 | 陰乘陰反而最吉 |
| 陰乘陽危 | 柔凌剛最凶 | ✗ 不成立 | 凶率並非最高 |

### 條件句數據庫關鍵發現

| 模式 | 出現次數 | 爻整體吉率 | 解讀 |
|------|----------|------------|------|
| 貞吉 | 60 | 96.7% | 「守正=吉」最可靠規則 |
| 貞凶 | 20 | 0.0% | 即使守正也凶=局勢極差 |
| 征凶 | 9 | 11.1% | 出征凶=應守不應攻 |
| 征吉 | 4 | 100% | 出征吉=時機成熟 |
| 无咎 | 85 | 25.9% | ≠吉，只是「及格」 |

### 位置×行動交互

| 位置 | 最佳行動 | 吉率 | 最差行動 | 吉率 |
|------|----------|------|----------|------|
| 初爻 | 貞 | 71% | - | - |
| 二爻 | 貞 | 91% | 征 | 25% |
| 三爻 | - | - | 貞 | 24% |
| 四爻 | 貞 | 71% | - | - |
| 五爻 | 往 | 100% | - | - |
| 上爻 | 征 | 50% | - | - |

### 核心洞見

1. **效應權重排序**：位置 >> 得中 >> 應爻 >> 當位 ≈ 承乘
2. **「貞」是萬金油**：多數位置守正都吉，除了三爻
3. **三爻是特殊位置**：連「貞」都只有24%吉率
4. **五爻適合主動**：「往」100%吉率，是最佳行動位置
5. **「无咎」重新定義**：不是「好」，而是「沒出錯」
