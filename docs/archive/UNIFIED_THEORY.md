
這就是為什麼：
- 乾卦（滿了）= 0% 吉率 → 沒錢了，沒得選
- 謙卦（空著）= 83% 吉率 → 銀行還有錢，隨時能動

## 三大原則

### 1. 空間原則：有餘地則吉

```
滿 = 凶    (乾卦6陽 = 0%)
適中 = 吉  (4陽爻 = 47.8%)
```

**為什麼？** 滿了就沒有發展餘地。4陽2陰 = 有陰的空間可以擴展。

### 2. 流動原則：能動則吉

```
卡住 = 凶  (三爻 = 10.9%)
能動 = 吉  (二五爻 = 46.9%)
```

**為什麼？** 三爻在下卦頂端，上不去下不來。二五爻在中間，可上可下。

### 3. 不對稱原則：有優勢則吉

```
體用平衡 = 凶  (25% 吉率)
體強用弱 = 吉  (37.5% 吉率)
```

**為什麼？** 平衡 = 無優勢 = 無法主導變化。有優勢 = 能主動選擇。

## 統一公式

```
吉凶 = f(變化空間)

變化空間 = 位置自由度 × 能量餘裕 × 關係優勢 × 流動可能
```

**簡化版：**
- 有空間 → 吉
- 被卡住 → 凶
- 太滿了 → 凶
- 能交流 → 吉

## 哲學意義

### 易 = 變

「易」字本義就是「變化」。

周易的核心智慧不是告訴你什麼是「好狀態」，而是告訴你：

> **能夠繼續變化的狀態，就是好狀態。**

這解釋了為什麼：
- 乾卦（最強）反而 0% 吉率 → 太強無法再強
- 謙卦（看似弱）卻是福地 → 有上升空間
- 三爻（中間）最凶 → 卡在轉折點
- 咸卦（感應）最吉 → 能與外界交流

## 實用指南

### 判斷吉凶的核心問題：

1. **還有空間嗎？** → 沒滿就好
2. **能動嗎？** → 不卡住就好
3. **有優勢嗎？** → 不用追求平衡
4. **能交流嗎？** → 感應比孤立好

### 生活應用：

- 事業到頂峰？→ 小心「亢龍有悔」
- 卡在中間層？→ 確實最危險
- 資源vs對手平衡？→ 要打破平衡，建立優勢
- 孤立無援？→ 找到「感應」的對象

## 結論

易經的智慧可以用一句話總結：

> **「吉」不是最好的狀態，而是最有發展可能的狀態。**

這就是為什麼：
- 最強的乾卦 ≠ 最吉
- 中間的謙卦 = 福地
- 平衡 ≠ 好
- 感應交流 = 最好的變化方向

## 圖論分析新發現

### 入口效應

「變到X卦的吉凶」≠「X卦本身的吉凶」

| 類型 | 例子 | 意義 |
|------|------|------|
| 難進但好 | 臨(自身83%, 變入0%) | 福地，但入口艱難 |
| 易進但差 | 既濟(自身0%, 變入67%) | 陷阱，進入容易但後果差 |

### 源卦比目標卦更重要

| 條件 | 變爻吉率 |
|------|----------|
| 從好卦變出 | 58.3% |
| 從差卦變出 | 10.9% |

**你從哪裡來，比你要去哪裡更重要。**

- **停留**：謙、臨、需（吸引子）
- **離開**：觀、恆、旅（排斥子）
- **警惕**：既濟、坎、困（陷阱）
