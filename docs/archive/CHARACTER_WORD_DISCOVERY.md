### 核心突破：字符-詞語範式

**類比思維：**
```
八卦 = 字符 (Character)
  - 8個基本符號 (0-7)
  - 每個代表抽象概念

卦 = 詞語 (Word)
  - 上字符 + 下字符 = 2字詞
  - 組合產生新意義

互卦 = 隱藏詞
  - 詞中嵌套的另一個詞
  - 影響整體傾向

這類似於：
- 中文：偏旁 + 偏旁 = 新字義
- DNA：codon (3 bases) = amino acid
- 化學：元素 + 元素 = 化合物
```

### 隨卦二爻之謎 - 已解

**問題：** 隨卦二爻結構完美（得中+有應+當位），為何實際為凶？

**解答：** 透過字符-詞語分析發現：
- 詞"31" = 兌上震下 = 悅+動
- 兌(3)在上位只有8%吉率 - 最差字符
- 這個位置效應蓋過了爻位優勢

```
公式表達：
隨卦二爻 = 得中(+0.5) + 兌在上(-0.4) + 震在下(+0.1) + 例外(-1.5)
         = -1.3 → 凶
```

4. **必然規則** - 發現100%成立的模式
5. **字符組合** - 八卦視為字符的關鍵突破
