# 爻義公式：從繫辭傳提取的核心規則

## 核心發現：繫辭下第九章

```
「二與四同功而異位，其善不同；二多譽，四多懼，近也。」
「三與五同功而異位，三多凶，五多功，貴賤之等也。」
「其柔危，其剛勝耶？」
```

這是《易經》中最接近「公式」的表述！

---

## 一、爻義預測公式

```
爻義 = 位置基礎值 + 陰陽修正 + 得位修正 + 中正修正 + 關係修正 × 時義係數
```

### 位置基礎值表

| 位置 | 基礎值 | 繫辭傳依據 | 說明 |
|------|--------|------------|------|
| 初 | 0 | 「其初難知」 | 潛藏，未定 |
| 二 | +2 | 「二多譽」 | 內卦之中，多讚譽 |
| 三 | -2 | 「三多凶」 | 內卦極點，多凶險 |
| 四 | -1 | 「四多懼」 | 近君之位，多憂懼 |
| 五 | +3 | 「五多功」 | 尊位，多功績 |
| 上 | -1 | 「其上易知，本末也」 | 終極，物極必反 |

### 陰陽修正

```
若陽爻(九)：+1 （「其剛勝」）
若陰爻(六)：-1 （「其柔危」）
```

### 得位修正

```
陽在陽位(初、三、五)：+1（剛居剛位，得位）
陰在陰位(二、四、上)：+1（柔居柔位，得位）
陽在陰位：0
陰在陽位：-1（柔居剛位，失位且危）
```

### 中正修正

```
二爻得中：+1
五爻得中：+1
中且正(六二、九五)：+2
中不正(九二、六五)：+1
```

### 關係修正（應、承、乘）

**應的修正：**
```
陰陽相應(初↔四, 二↔五, 三↔上)：+1
同性無應：0
應而敵(同性對抗)：-1
```

**承乘修正：**
```
陰承陽(柔順承剛)：+1
陽承陰(剛承柔，不順)：0
陰乘陽(柔凌剛)：-1
陽乘陰(正常)：0
```

### 時義係數

```
若爻義符合卦之時義：×1.5
若爻義違背卦之時義：×0.5
中性：×1
```

## 二、公式應用示例

### 示例1：乾卦九五「飛龍在天，利見大人」

```
位置：五（+3）
陰陽：陽（+1）
得位：九在五，陽在陽位（+1）
中正：五爻得中且正（+2）
相應：與九二，同性無應（0）
承乘：乘九四，陽乘陽（0）
時義：乾之時為剛健，九五正顯剛健（×1.5）

爻值 = (3+1+1+2+0+0) × 1.5 = 10.5 → 大吉
```

**驗證**：九五「飛龍在天」確實是乾卦最吉之爻。✅

### 示例2：履卦六三「眇能視，跛能履，履虎尾，咥人，凶」

```
位置：三（-2）
陰陽：陰（-1）
得位：六在三，陰在陽位（-1）
中正：三爻不中（0）
相應：與上九，陰陽相應（+1）
承乘：乘九二，陰乘陽（-1）
時義：履之時為謹慎禮儀，六三冒進（×0.5）

爻值 = (-2-1-1+0+1-1) × 0.5 = -2 → 凶
```

**驗證**：六三「履虎尾，咥人，凶」確實是履卦最凶之爻。✅

### 示例3：謙卦九三「勞謙，君子有終，吉」

```
位置：三（-2，本凶位）
陰陽：陽（+1）
得位：九在三，陽在陽位（+1）
中正：三爻不中（0）
相應：與上六，陰陽相應（+1）
承乘：乘六二，陽乘陰（0）
時義：謙之時為謙卑，九三「勞謙」正符合（×1.5）

爻值 = (-2+1+1+0+1+0) × 1.5 = 1.5 → 吉
```

**驗證**：九三「勞謙，君子有終，吉」是三爻中罕見的吉例，因時義救了危位。✅

## 三、從公式推導的新發現

### 發現1：位置-陰陽組合的吉凶傾向

| 組合 | 計算值 | 傾向 |
|------|--------|------|
| 九五 | 3+1+1+2=7 | 極吉 |
| 六二 | 2-1+1+2=4 | 大吉 |
| 九二 | 2+1+0+1=4 | 吉 |
| 六五 | 3-1+0+1=3 | 吉（柔中） |
| 初九 | 0+1+1+0=2 | 小吉 |
| 六四 | -1-1+1+0=-1 | 小凶 |
| 九四 | -1+1+0+0=0 | 中性 |
| 上九 | -1+1+0+0=0 | 中性 |
| 九三 | -2+1+1+0=0 | 中性（需時義救）|
| 初六 | 0-1-1+0=-2 | 小凶 |
| 上六 | -1-1+1+0=-1 | 小凶 |
| 六三 | -2-1-1+0=-4 | 極凶 |

**結論**：六三是最危險的組合（陰爻在內卦極點的陽位）

### 發現2：「同功異位」的對比模式

繫辭說「二與四同功，三與五同功」，實際計算：

| 同功對 | 二/四差異 | 三/五差異 |
|--------|-----------|-----------|
| 陽爻 | 九二(4) vs 九四(0) = 差4 | 九三(0) vs 九五(7) = 差7 |
| 陰爻 | 六二(4) vs 六四(-1) = 差5 | 六三(-4) vs 六五(3) = 差7 |

**結論**：三/五的差異(7)大於二/四的差異(4-5)，這符合「貴賤之等也」的描述。

### 發現3：時義的「逆轉」能力

時義係數可以將：
- 本凶(-2)變吉(+1.5)：如謙卦九三
- 本吉(+4)變凶：當爻義違背卦旨時

**這解釋了為什麼同樣的位置在不同卦中吉凶不同。**

## 五、公式的應用價值

1. **預測爻辭吉凶**：輸入位置和結構，輸出吉凶傾向
2. **解釋例外**：當計算值與實際不符時，找出是哪個因素造成差異
3. **深度解讀**：不只是「這爻吉」，而是「因為X+Y+Z所以吉」
4. **發現規律**：統計分析可以找出古人沒有明說的模式
