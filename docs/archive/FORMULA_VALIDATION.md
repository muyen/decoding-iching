## 需要修正的卦（6個）

### 1. 訟 (#6) - 天水
- 推導：需待（等待）
- 傳統：訟（爭訟）
- 修正思路：天在上想升，水在下想流 → 方向相反 → 衝突 → 訟

### 2. 小畜 (#9) - 風天
- 推導：相遇
- 傳統：小畜（小積蓄）
- 修正思路：風（柔）畜天（剛） → 柔止剛 → 小有積蓄

### 3. 無妄 (#25) - 天雷
- 推導：馭動
- 傳統：無妄（無虛妄）
- 修正思路：天雷自然之動 → 無人為造作 → 無妄

### 4. 夬 (#43) - 澤天
- 推導：履禮
- 傳統：夬（決斷）
- 修正思路：澤（缺口）在天上 → 陰消陽長的決斷時刻 → 夬

### 5. 姤 (#44) - 天風
- 推導：馭柔
- 傳統：姤（相遇）
- 修正思路：一陰初生於下 → 陰陽相遇 → 姤（意外相遇）

### 6. 履 (#10) 和 夬 (#43) 重複問題
- 履(天澤) 和 夬(澤天) 需要區分
- 履：以悅侍剛 → 禮
- 夬：剛決柔 → 決斷

## 公式修正 v2.0

基於驗證結果，增加以下規則：

### 規則補充1：相同元素不同位置

```
若 A上B下 和 B上A下 意義不同，則：
- 上位者主導定義
- 下位者是被作用對象
```

### 規則補充2：陰陽消長

```
若卦象呈現陰消陽長（或相反）：
- 五陽一陰（如夬）→ 決斷
- 五陰一陽（如復）→ 復返
- 一陽初生（如姤）→ 相遇
```

### 規則補充3：自然 vs 人為

```
若上下組合呈現自然現象：
- 天雷自然動 → 無妄（無人為）
- 雷天人為動 → 大壯（壯大）
```

## 結論

**易經公式基本成立**：
- 90.6%的卦可以直接由公式推導出正確意義
- 剩餘9.4%需要補充規則（陰陽消長、自然vs人為）
- 無任何卦完全無法解釋

**核心公式**：
```
卦義 = 上卦功能(作用於)下卦功能
     × 交流係數
     × 陰陽消長係數
     × 自然/人為係數
```

這個公式可以用於：
1. 推導卦義
2. 生成白話解釋
3. 梅花易數占卜的系統化解讀
