## 最重要的發現

### 1. 八卦像「字母」，六十四卦像「單詞」

```
八卦 = 8個字母 (0-7)
六十四卦 = 兩個字母組成的單詞

例如：
  乾卦 = "77" (乾上乾下)
  坤卦 = "00" (坤上坤下)
  謙卦 = "04" (坤上艮下)
```

這個「字母-單詞」的想法，讓我們理解為什麼某些卦天生就比較吉或凶。
