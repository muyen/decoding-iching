### 2.1 六爻基礎值

| 爻位 | 基礎值 | 繫辭傳依據 | 統計驗證 |
|------|--------|------------|----------|
| 初 | 0 | 「其初難知」 | 「吉」13.2%，「凶」10.5% |
| 二 | +2 | 「二多譽」 | 「吉」20.8%，「凶」7.9%（最少） |
| 三 | -2 | 「三多凶」 | 「吉」10.4%（最少），「凶」28.9%（最多） |
| 四 | -1 | 「四多懼」 | 「无咎」22.2%（最多！）→ 補救之位 |
| 五 | +3 | 「五多功」 | 「吉」23.6%（最多），「凶」7.9%（最少） |
| 上 | -1 | 「其上易知」 | 「凶」28.9%（與三並列），但「悔亡」最多 |

### 2.2 重大修正發現

**發現一：四爻是「補救之位」而非單純「凶位」**
- 傳統觀點：四爻多懼
- 統計發現：四爻「无咎」比例最高（22.2%）
- 新解讀：四爻是最能通過謹慎行為化險為夷的位置

**發現二：上爻有兩種結局**
- 傳統觀點：物極必反
- 統計發現：上爻「悔亡」8次（最多），「終吉」7次
- 新解讀：上爻可凶可吉，取決於是否及時醒悟

## 第四部分：四維關係網絡

### 4.1 錯卦（Complement）

**定義**：陰陽全反的兩卦

**語義規律**：
- 互補而非對立
- 如乾坤：創造vs承載，不是好vs壞
- 雜卦傳完美對應錯卦關係

### 4.2 綜卦（Inverse）

**定義**：上下顛倒的兩卦

**爻位對應**：
- 本卦初爻 ↔ 綜卦上爻
- 本卦二爻 ↔ 綜卦五爻
- 本卦三爻 ↔ 綜卦四爻

**King Wen Pairs語義相似度**：0.7565

### 4.3 互卦（Nuclear）

**定義**：取2-3-4爻為下，3-4-5爻為上

**影響力**：0.7315（最強！基線僅0.1930）

**規律**：
- 互卦揭示本卦「內在本質」
- 爻辭常暗示互卦意象
- 泰卦六五「帝乙歸妹」，互卦即為歸妹

### 4.4 變卦

**定義**：單爻變動後的新卦

**規律**：
- 凶爻變卦多為警示
- 吉爻變卦呈現發展方向

## 第五部分：應與承乘量化

### 5.1 應的統計效應

| 應位類型 | 有應吉類佔比 | 無應吉類佔比 | 差異 |
|----------|--------------|--------------|------|
| 二↔五 | **68.8%** | 51.6% | +17.2% |
| 初↔四 | 46.9% | 43.8% | +3.1% |
| 三↔上 | 34.4% | 28.1% | +6.3% |

**結論**：二五相應效果最強，三上相應效果最弱

### 5.2 承乘統計效應

| 承乘類型 | 吉佔比 | 凶佔比 | 傳統說法驗證 |
|----------|--------|--------|--------------|
| 陽乘陰 | **43.8%** | 25.0% | ✓ 剛制柔正常 |
| 陰承陽 | 35.0% | 35.0% | △ 不完全成立 |
| 陰乘陽 | 22.5% | **50.0%** | ✓ 柔凌剛不吉 |

**陰乘陽轉吉條件**：得中 × 得尊位 × 合時義（三者具二即可）

### 7.1 五大解釋模式

| 模式 | 結構 | 示例 |
|------|------|------|
| 因果 | 以[原因]也 | 「利用刑人，以正法也」 |
| 位置 | 位正當/不當 | 「咥人之凶，位不當也」 |
| 時機 | 與時行/時舍 | 「終日乾乾，與時偕行」 |
| 德行 | 志行正/穷 | 「虽磐桓，志行正也」 |
| 關係 | 乘刚/合志 | 「六二之難，乘刚也」 |

### 7.2 志向判斷系統

| 志向狀態 | 含義 | 預測 |
|----------|------|------|
| 志行正也 | 志向正當且可行 | 吉 |
| 志在外也 | 志向在外發展 | 宜進取 |
| 志未得也 | 志向未能實現 | 需等待 |
| **志穷也** | **志向走投無路** | **凶** |
| 志大行也 | 志向大為施行 | 大吉 |

### 7.3 「未」字預測法

| 用法 | 含義 | 預測 |
|------|------|------|
| 未失常也 | 還沒失去常道 | 可無咎 |
| 未光也 | 還沒光大 | 有上升空間 |
| 未亡也 | 還沒滅亡 | **有希望** |

**規則**：「未」字表示「尚有轉機」，不是終局判斷。

## 第九部分：深度解釋七步法

### Step 1：確定卦之時義類型

```
順時卦(16卦) → 時義係數 +1.5
逆時卦(14卦) → 時義係數 -0.5~-1.5
轉換卦(16卦) → 時義係數 ±1.0
等待卦(18卦) → 時義係數 +1.0
```

### Step 2：計算爻位基礎值

```
初爻=0, 二爻=+2, 三爻=-2, 四爻=-1, 五爻=+3, 上爻=-1
```

### Step 3：判斷得位與得中

```
得位（陽居陽/陰居陰）：+1
失位（陽居陰/陰居陽）：0或-1
得中（二爻或五爻）：+1
中且正（六二或九五）：+2
```

### Step 4：分析應與承乘

```
二五有應：+0.5（效果最強）
初四有應：+0.3
三上有應：+0.1
陰乘陽：-0.7
陽乘陰：+0.3
```

### Step 5：查詢四維關係

```
錯卦：提供互補視角
綜卦：提供鏡像對照
互卦：揭示內在本質（最重要，0.7315影響力）
變卦：預示發展方向
```

### Step 6：對照象傳解讀模式

```
查看志向狀態：志行正？志穷？
查看位置評價：位正當？位不當？
查看「未」字線索：有轉機？
```

### Step 7：綜合判斷並生成解釋

```
爻義分數 = (基礎值 + 各修正) × 時義係數

分數 > 5：大吉
分數 3~5：吉/無不利
分數 1~3：小吉/無咎
分數 0~1：吝/厲
分數 < 0：凶
```

## 第十部分：新發現總結

### 10.1 挑戰傳統的發現

| 傳統觀點 | 數據證據 | 修正觀點 |
|----------|----------|----------|
| 守正必吉 | 「貞凶」出現8次 | 守正需要時機 |
| 四爻多懼 | 四爻无咎最多(22.2%) | 四爻是補救之位 |
| 上爻必悔 | 上爻終吉7次，悔亡8次 | 上爻有選擇 |
| 當位重要 | 既濟全當位卻「終亂」 | 當位影響最小(3%) |
| 利見大人適用所有 | 只出現在五爻和卦辭 | 需要相應地位 |

### 10.2 前所未有的模式

1. **四爻是補救之位**：通過謹慎行為可化險為夷
2. **互卦影響力最強**：0.7315 vs 0.1930基線
3. **時義可完全逆轉位置吉凶**：謙卦六爻皆吉
4. **數字有固定象徵**：三=週期，七=回歸，十=不可能
5. **身體意象完美對應爻位**：趾→初，首→上

### 10.3 優先級重排

**傳統強調順序**：當位 > 得中 > 應 > 承乘

**統計驗證順序**：**時義 > 爻位 > 得中 > 應 > 承乘 > 當位**
