# 易經公式：大道至簡

## 目標

如同牛頓的 F = ma 可以預測物理行為，我們要找出易經的核心公式，使得：

```
卦義 = f(結構參數)
```

---

## 觀察：核心變量

從說卦傳和64卦分析中，我們識別出以下核心變量：

### 變量1：八卦功能向量

每個三畫卦有一個**核心功能**（動詞）：

| 卦 | 功能 | 數值化描述 |
|----|------|------------|
| 乾 | 君(主導) | +3 陽性、主動、向上 |
| 坤 | 藏(收納) | -3 陰性、被動、向下 |
| 震 | 動(啟動) | +2 陽性、開始 |
| 巽 | 入(滲透) | -1 陰性、進入 |
| 坎 | 陷(危險) | -2 陰性、下沉 |
| 離 | 麗(附著) | +1 陽性、顯現 |
| 艮 | 止(停止) | +1 陽性、穩定 |
| 兌 | 說(喜悅) | -1 陰性、開放 |

### 變量2：位置（上/下）

```
上卦 = 外在、結果、環境、未來、他人
下卦 = 內在、起因、自身、過去、自己
```

### 變量3：交流係數

```
交流 = 上下是否互動

若上卦「想下」且下卦「想上」→ 交流 = +1 (通)
若上卦「想上」且下卦「想下」→ 交流 = -1 (塞)
若同向或中性 → 交流 = 0 (普通)
```

### 變量4：陰陽平衡

```
陽爻數 = 0-6
平衡度 = |陽爻數 - 3| (越接近3越平衡)
```

## 公式假設 v1.0

### 核心公式

```
卦義 = 關係類型 × 交流係數

其中：
關係類型 = 上卦功能 ⊕ 下卦功能
交流係數 = 基於能量流向的判定
```

定義「⊕」為功能組合運算：

| 上＼下 | 乾(君) | 坤(藏) | 震(動) | 巽(入) | 坎(陷) | 離(麗) | 艮(止) | 兌(說) |
|--------|--------|--------|--------|--------|--------|--------|--------|--------|
| **乾** | 純剛 | 不交 | 馭動 | 馭入 | 訟險 | 大有 | 遁止 | 決說 |
| **坤** | 交泰 | 純柔 | 復動 | 升入 | 師險 | 晉麗 | 謙止 | 臨說 |
| **震** | 大壯 | 豫動 | 純動 | 恆入 | 解陷 | 豐麗 | 小過 | 歸妹 |
| **巽** | 姤柔 | 觀藏 | 益動 | 純入 | 渙散 | 家人 | 漸止 | 中孚 |
| **坎** | 需險 | 比藏 | 屯動 | 井入 | 純陷 | 既濟 | 蹇阻 | 困說 |
| **離** | 同人 | 明夷 | 噬嗑 | 鼎入 | 未濟 | 純麗 | 旅止 | 睽說 |
| **艮** | 大畜 | 剝藏 | 頤養 | 蠱腐 | 蒙蔽 | 賁飾 | 純止 | 損說 |

## 公式假設 v2.0：更簡化

### 終極簡化公式

```
卦義 = 上功能(作用於)下功能 + 位置修正

位置修正：
- 若違反自然位置(如地上天下) → 交流、通
- 若符合自然位置但不交流 → 閉塞、否
- 若功能相生 → 吉
- 若功能相克 → 凶或挑戰
```

### 用自然語言表達

```
[上卦代表的力量] 對 [下卦代表的狀態] 產生作用
= 卦的核心情境
```

**例子**：

| 卦 | 上 | 下 | 公式展開 | 結果 |
|----|----|----|----------|------|
| 泰 | 地(藏↓) | 天(君↑) | 藏欲下+君欲上=交流 | 通泰 |
| 否 | 天(君↑) | 地(藏↓) | 君欲上+藏欲下=背離 | 閉塞 |
| 屯 | 水(陷) | 雷(動) | 動遇險=初生之難 | 艱難 |
| 蒙 | 山(止) | 水(陷) | 止於險前=看不清 | 蒙昧 |
| 需 | 水(險) | 天(剛) | 剛遇險=等待 | 等待 |
| 訟 | 天(剛) | 水(險) | 剛在險上=爭訟 | 爭訟 |

## 驗證公式

### 測試案例1：師 ䷆ (地水師)

```
上：坤(藏、眾)
下：坎(險、水)

公式：眾人藏於險中 = 軍隊
結果：師（軍隊）✅
```

### 測試案例2：比 ䷇ (水地比)

```
上：坎(水、親)
下：坤(地、眾)

公式：水親近大地 = 親比
結果：比（親近）✅
```

### 測試案例3：小畜 ䷈ (風天小畜)

```
上：巽(風、柔)
下：乾(天、剛)

公式：柔風積於剛天 = 小有積蓄
結果：小畜（小積蓄）✅
```

### 測試案例4：履 ䷉ (天澤履)

```
上：乾(天、剛、尊)
下：兌(澤、悅、卑)

公式：以悅(卑)侍剛(尊) = 禮儀
結果：履（禮）✅
```

## 公式的數學表達（嘗試）

設：
- T = 上卦向量 (功能, 方向, 強度)
- B = 下卦向量 (功能, 方向, 強度)
- I = 交互作用函數

則：

```
卦義向量 = I(T, B)

其中 I 包含：
1. 功能組合：T.func ⊗ B.func
2. 方向交互：T.dir × B.dir (同向=-1, 逆向=+1)
3. 強度調節：(T.str + B.str) / 2
```

## 發現：三個核心規則

經過推導，我認為易經系統可以歸結為三條核心規則：

### 規則1：功能組合

```
卦義 = 上卦功能 ⊕ 下卦功能
```

每個功能組合產生一個特定情境。

### 規則2：位置交流

```
交流係數 = sign(上卦方向 × 下卦方向 × -1)

若 上想下 且 下想上 → 交流(+)
若 上想上 且 下想下 → 閉塞(-)
否則 → 中性(0)
```

### 規則3：卦名來自核心意象

```
卦名 = 核心意象的最簡表達

如：地天交流 → 泰(通)
    天地背離 → 否(塞)
    水雷初生 → 屯(難)
```
