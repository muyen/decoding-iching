## 新發現：卦的三態模型

### 概念
每個卦像物質一樣，可以處於三種狀態：
- **吉態** (類似固態) - 穩定、積極
- **中態** (類似液態) - 流動、中性
- **凶態** (類似氣態) - 不穩定、消極

## 核心發現：卦的交互模型

### 關鍵洞察：卦本身是中性的

**類比**：卦像火一樣 - 火本身無所謂好壞，只有當它與其他事物交互時，才產生吉凶。

### 三元卦極性（Polarity）分析

| 三元卦 | 作為內卦 | 作為外卦 | 極性差 | 自然現象對應 |
|--------|----------|----------|--------|--------------|
| **震** | **-0.125** | **+1.375** | **+1.5** | 雷在地下=地震=凶；雷在天上=自然=吉 |
| 乾 | +0.5 | +1.0 | +0.5 | 天在下=不自然；天在上=順應 |
| 坎 | 0.0 | +0.5 | +0.5 | 水在下=停滯；水在上=流動 |
| 離 | +0.5 | +0.75 | +0.25 | 火無論何處都有能量 |
| 坤 | +0.625 | +0.75 | +0.125 | 地厚載物，位置影響小 |
| 艮 | +0.125 | +0.25 | +0.125 | 山止也，位置影響最小 |
| 巽 | +0.625 | +0.625 | 0.0 | 風無形，位置不影響 |
| 兌 | +0.625 | +0.5 | -0.125 | 澤潤萬物，內外皆宜 |

### 發現：古人選擇的象徵完美匹配統計行為！

**震（Thunder）的極性最大（+1.5）**：
- 作為內卦（下卦）：-0.125（負面）
- 作為外卦（上卦）：+1.375（非常正面）
- 這完全對應自然現象：
  - 雷在地下 = 地震 = 災難
  - 雷在天上 = 正常雷暴 = 自然

**艮（Mountain）的極性最小（+0.125）**：
- 艮代表「止」
- 無論在上在下，山都是穩定的
- 這也符合山的特性 - 不會因位置而改變本質

### 交互反應係數

每個卦在不同爻位有不同的「反應係數」，決定該位置是否容易產生吉凶變化：

```
敏感位置（容易變化）：reaction_coefficient ≠ 0
穩定位置（不易變化）：reaction_coefficient = 0
```

**最敏感的卦**（所有位置都敏感）：
- 頤 (27)：inner_outer_diff = 6（內外差異最大）

**最穩定的卦**（所有位置都穩定）：
- 乾 (1)：全部爻位係數為0
- 觀 (20)：全部爻位係數為0
