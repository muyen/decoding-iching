# 卦象關係完整指南：六種變化關係的實用解讀

*整理日期：2026-01-21*

---

## 總覽：六種關係的角色

| 關係 | 定義 | 作用 | 何時用 |
|------|------|------|--------|
| **應** | 初↔四、二↔五、三↔上 | 看有沒有外援 | 問合作、求助 |
| **承乘** | 相鄰兩爻上下關係 | 看上下關係順不順 | 問職場、輩分 |
| **互卦** | 爻2-5組成的卦 | 看內在本質（最重要！）| 看深層走向 |
| **錯卦** | 陰陽全反 | 看互補面 | 看對立統一 |
| **綜卦** | 上下顛倒 | 看對方視角 | 問感情、談判 |
| **變卦** | 一爻變動後的卦 | 看發展趨勢 | 占得動爻時 |

---

## 一、應（對應爻）

### 定義
```
初爻 ←→ 四爻
二爻 ←→ 五爻  ← 最重要（君臣相應）
三爻 ←→ 上爻
```

### 規則
- 一陰一陽 = **有應**（互相呼應）
- 兩陽或兩陰 = **無應**（各自為政）

### 統計發現

| 應位 | 有應吉率 | 無應吉率 | 效果 |
|------|----------|----------|------|
| **二↔五** | **68.8%** | 51.6% | **+17.2%（最強）** |
| 初↔四 | 46.9% | 43.8% | +3.1%（弱） |
| 三↔上 | 34.4% | 28.1% | +6.3%（弱） |

### 實用意義
- **問事業**：看二五應（你與老闆/貴人的關係）
- **二五有應**：有人幫、資源到位
- **二五無應**：需自力更生

---

## 二、承乘

### 定義
```
承 = 下面的爻「承托」上面的爻
乘 = 上面的爻「騎乘」下面的爻
```

### 四種組合及效果

| 類型 | 吉率 | 凶率 | 意義 |
|------|------|------|------|
| 陽乘陰 | **43.8%** | 25.0% | 剛統柔，最順 |
| 陰承陽 | 35.0% | 35.0% | 柔順剛，平衡 |
| 陽承陰 | 中等 | 中等 | 剛在柔下，委屈 |
| **陰乘陽** | 22.5% | **50.0%** | **柔凌剛，最危** |

### 化解「陰乘陽」的條件
```
得中 + 得尊位 + 合時義 → 三者具二即可轉吉
```

### 實用意義
- **問職場**：看你與上司的陰陽關係
- 陰（柔）在陽（剛）上 = 下屬壓過上司，需格外謹慎

---

## 三、互卦（最重要！）

### 定義
```
取爻 2, 3, 4 → 互卦下卦
取爻 3, 4, 5 → 互卦上卦
```

### 統計發現
- **影響力**：0.7315（其他關係只有 0.19 基線）
- **本卦與互卦吉凶一致率**：48.4%

### 作用
- 揭示事物的**內在本質**
- 爻辭常暗示互卦意象
- 例：泰卦六五「帝乙歸妹」，互卦即為歸妹

### 實用意義
- **必看互卦**：了解表象背後的真相
- 本卦吉但互卦凶 = 表面好但有隱患
- 本卦凶但互卦吉 = 困難中有出路

---

## 四、錯卦

### 定義
所有爻陰陽互換（如乾↔坤）

### 統計發現
- **吉凶相反率**：只有 28.1%
- 並非「對立」而是「互補」

### 作用
- 提供**互補視角**
- 乾坤：創造 vs 承載（不是好 vs 壞）
- 泰否：通泰 vs 閉塞（互為因果）

### 實用意義
- **問複雜問題**：看錯卦理解另一面
- 事物總有互補的另一面存在

---

## 五、綜卦

### 定義
整卦上下顛倒（180度翻轉）

### 統計發現
- **吉凶相近率**：**64.3%（最高相似度）**
- 8卦自對稱（乾、坤、頤、大過、坎、離、中孚、小過）

### 爻位對應
```
本卦初爻 ↔ 綜卦上爻
本卦二爻 ↔ 綜卦五爻
本卦三爻 ↔ 綜卦四爻
```

### 作用
- 提供**不同立場**看同一事
- 把卦倒過來 = 換個角度思考

### 實用意義
- **問感情/談判**：把卦倒過來看對方視角
- 你是初爻，對方就是上爻的處境
- 理解對方後更容易找到共識

---

## 六、變卦（之卦）

### 定義
一爻變動（陰↔陽）後形成的新卦

### 統計發現（384種變化分析）

| 指標 | 數值 |
|------|------|
| 之卦趨勢向上 (↑) | 148 (38.5%) |
| 之卦趨勢向下 (↓) | 148 (38.5%) |
| 之卦趨勢持平 (→) | 88 (22.9%) |

### 重要發現

**「凶爻變吉」案例（有出路）**：26例
| 本卦 | 動爻 | 之卦 | 趨勢 |
|------|------|------|------|
| 師 | 初爻 | 臨 | +0.67 |
| 恆 | 上爻 | 鼎 | +0.67 |
| 乾 | 三爻 | 履 | +0.50 |

**「吉爻變凶」案例（好景不長）**：64例
| 本卦 | 動爻 | 之卦 | 趨勢 |
|------|------|------|------|
| 臨 | 五爻 | 節 | -0.83 |
| 比 | 二爻 | 坎 | -0.67 |
| 鼎 | 上爻 | 恆 | -0.67 |

### 實用意義
- **占得動爻時必看變卦**
- 動爻凶但之卦好 = 窮則變，變則通
- 動爻吉但之卦差 = 居安思危，盛極必衰

## 解卦優先順序

### 權重排序（經統計驗證）

```
1. 時義 (40%)     ← 卦的整體主題
2. 爻位 (25%)     ← 你在什麼位置
3. 得中 (15%)     ← 是否二或五
4. 應 (10%)       ← 有沒有外援
5. 承乘 (7%)      ← 上下關係順不順
6. 當位 (3%)      ← 陰陽是否得位
```

### 完整解卦公式

```
完整解讀 = 本卦（現狀）
         + 動爻（轉折點）
         + 之卦（趨勢）
         + 互卦（內在本質）
         + 應爻（外援）
         + 綜卦（對方視角，選用）
```

## 實際應用範例

### 問事業發展

1. **先看本卦時義**：順時卦（有利）還是逆時卦（困難）？
2. **看爻位**：你在初（起步）、二五（核心）、還是上（結束）？
3. **看二五應**：與上司/貴人關係如何？
4. **看互卦**：內在發展趨勢是什麼？
5. **有動爻則看變卦**：事情會往哪個方向發展？

### 問感情關係

1. **看本卦**：關係的現狀
2. **看綜卦**：把卦倒過來，理解對方的處境
3. **看互卦**：這段關係的本質
4. **看變卦**：關係會如何發展

### 問是否行動

1. **看本卦時義**：現在是動的時機嗎？
2. **看動爻吉凶**：行動的這一步好不好？
3. **看變卦**：行動後會變成什麼局面？
4. **比較本卦與變卦**：變了比不變好嗎？

## 總結

| 問題類型 | 主要看 | 輔助看 |
|----------|--------|--------|
| 現狀分析 | 本卦、爻位 | 互卦 |
| 趨勢預測 | 變卦、互卦 | 時義 |
| 關係問題 | 應、綜卦 | 承乘 |
| 決策判斷 | 時義、變卦 | 爻位 |

**核心原則**：
1. 時義最重要（40%），可以逆轉一切
2. 互卦揭示真相（影響力最強 0.73）
3. 變卦看趨勢（38.5%會變好或變差）
4. 綜卦看對方（64.3%相似度）
