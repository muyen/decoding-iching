# 易經384爻 修正後分析報告

## 摘要

本報告基於**修正後的爻辭標籤**進行分析。與之前報告的主要差異：

| 指標 | 之前 | 修正後 | 差異原因 |
|------|------|--------|----------|
| 吉率 | 35.9% | 34.6% | - |
| 中率 | 37.8% | **48.7%** | 修正「无不利」誤分類 |
| 凶率 | 26.3% | **16.7%** | 修正「无不利」誤分類 |
| 基線 | 37.8% | **48.7%** | 多數類為「中」|

### 修正內容

1. **「无不利」分類錯誤**：之前被誤分為「凶」（因為包含「不利」），現修正為「吉」
2. **爻辭提取錯誤**：部分卦（如咸卦）使用`，`而非`：`作為分隔符
3. **卦52-64結構問題**：這些卦的content包含彖傳和象傳，需用正則表達式提取爻辭

## 六、陰陽「當位」分析

傳統理論認為「陽爻居奇位、陰爻居偶位」為當位。

| 當位 | 總數 | 吉率 | 凶率 |
|------|------|------|------|
| 當位 | 192 | 33.3% | 15.6% |
| 不當位 | 192 | 35.9% | 17.7% |

**結論**：整體而言，「當位」與「不當位」差異不大。

但**特定位置有顯著差異**：

| 爻位 | 陽爻吉率 | 陰爻吉率 | 陽爻凶率 | 陰爻凶率 |
|------|----------|----------|----------|----------|
| 初爻 | **43.8%** | 34.4% | 6.2% | **25.0%** |
| 二爻 | 43.8% | **46.9%** | 12.5% | 9.4% |
| 三爻 | **15.6%** | 6.2% | 34.4% | 31.2% |
| 四爻 | **40.6%** | 28.1% | 12.5% | **3.1%** |
| **五爻** | 40.6% | **56.2%** | 6.2% | 6.2% |
| 上爻 | **34.4%** | 25.0% | 18.8% | **34.4%** |

**關鍵發現**：
- **五爻（君位）陰爻更吉**（56.2% vs 40.6%）- 柔順治國
- **初爻陽爻更吉**（43.8% vs 34.4%）- 剛健起步
- **上爻陰爻更凶**（34.4%凶）- 柔極必敗

## 八、結論

### 已驗證的傳統智慧

1. **「九五之尊」** ✓ - 五爻確實最吉
2. **「三多凶」** ✓ - 三爻確實最凶
3. **「二多譽」** ✓ - 二爻次吉
4. **「亢龍有悔」** ✓ - 上爻吉凶接近
5. **「坎為險」** ✓ - 坎卦吉率最低

### 新發現

1. **中層（3-4爻）最危險**：不僅三爻凶，整個中層都是風險區
2. **相生優於比和**：五行比和反而不吉
3. **非對稱卦更吉**：變化帶來機會
4. **五爻陰優於陽**：柔順治國勝過剛健

### 預測極限

- **基線準確率**：48.7%（猜「中」）
- **結構預測極限**：~50%（經交叉驗證確認）
- **結論**：純結構只能預測一半，另一半需要閱讀爻辭
