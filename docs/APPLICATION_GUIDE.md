## 二、起卦方法

### 2.1 時間起卦（最常用）

```
年數 + 月數 + 日數 = 上卦數
年數 + 月數 + 日數 + 時辰數 = 下卦數
總數 ÷ 6 的餘數 = 動爻

年數：按地支序號（子1、丑2...亥12）
時辰：子1、丑2、寅3...亥12

卦數對應（先天八卦數）：
1=乾、2=兌、3=離、4=震、5=巽、6=坎、7=艮、8=坤
（若餘數為0，則取8=坤）
```

**範例**：2026年1月20日午時
- 年：2026年為午年 = 7
- 月：1月 = 1
- 日：20日 = 20
- 時：午時 = 7

上卦：(7+1+20) ÷ 8 = 28 ÷ 8 = 3餘4 → 震
下卦：(7+1+20+7) ÷ 8 = 35 ÷ 8 = 4餘3 → 離
動爻：35 ÷ 6 = 5餘5 → 第五爻動

本卦：雷火豐（震上離下），五爻動，變卦：地火明夷

### 2.2 數字起卦

```
任意取兩個數字：
- 第一數 ÷ 8 餘數 = 上卦
- 第二數 ÷ 8 餘數 = 下卦
- 兩數之和 ÷ 6 餘數 = 動爻

適用：看到數字（車牌、門牌、電話等）
```

### 2.3 報數起卦

```
請問卦者隨意報一個數字：
- 若報一個數：此數為上卦，當前時辰為下卦
- 若報兩個數：分別為上下卦
- 若報三個數：前二數為卦，第三數定動爻
```

### 2.4 文字起卦（測字）

```
取字的筆畫數：
- 單字：筆畫數 ÷ 8 = 上卦，配時辰為下卦
- 兩字：各自筆畫分別為上下卦
- 多字：前半筆畫和為上卦，後半為下卦

動爻：總筆畫 ÷ 6 的餘數
```

## 三、解卦流程

### 3.1 標準流程

```
步驟 1：確定本卦
        ↓
步驟 2：確定動爻與變卦
        ↓
步驟 3：確定體用
        - 動爻所在的卦為「用」
        - 不動的卦為「體」
        ↓
步驟 4：分析體用五行生克
        ↓
步驟 5：查互卦（中間四爻組成）
        - 2、3、4爻為下互卦
        - 3、4、5爻為上互卦
        ↓
步驟 6：考慮卦氣旺衰（季節）
        ↓
步驟 7：看外應（環境信號）
        ↓
步驟 8：綜合判斷
```

### 3.2 體用判定

```
關鍵原則：動爻所在的卦為「用」，不動的卦為「體」

例：雷火豐，五爻動
- 上卦震（4、5、6爻），第五爻動 → 震為用
- 下卦離（1、2、3爻），不動 → 離為體

體：離（火）
用：震（木）
關係：用生體（木生火）→ 吉
```

### 3.3 互卦計算

```
本卦六爻：
    6爻 ─
    5爻 ─  ← 上互卦上爻
    4爻 ─  ← 上互卦中爻、下互卦上爻
    3爻 ─  ← 上互卦下爻、下互卦中爻
    2爻 ─  ← 下互卦下爻
    1爻 ─

互卦 = 上互卦 + 下互卦（用3、4、5爻為上，2、3、4爻為下）

互卦代表事情發展的中間過程。
```

## 五、應期推斷

### 5.1 基本公式

```
應期 = f(成卦之數, 動靜狀態, 卦氣旺衰, 體用關係)

成卦之數 = 起卦時的總數

動靜調整：
- 行走中問卦：成卦之數 ÷ 2
- 站立中問卦：成卦之數
- 坐著問卦：成卦之數 × 2
```

### 5.2 單位判斷

| 事情性質 | 適用單位 |
|----------|----------|
| 即時小事 | 時辰 |
| 日常事務 | 日 |
| 中期計劃 | 月 |
| 長期大事 | 年 |

```
輔助判斷：
- 動卦（震巽）：應期較快
- 靜卦（艮坤）：應期較慢
- 用生體：應期快
- 用克體：應期慢或不成
```

### 5.3 卦數應期

| 卦 | 先天數 | 可能應期 |
|----|--------|----------|
| 乾 | 1 | 1日/1月/初一/十一 |
| 兌 | 2 | 2日/2月/初二/十二 |
| 離 | 3 | 3日/3月/初三/十三 |
| 震 | 4 | 4日/4月/初四/十四 |
| 巽 | 5 | 5日/5月/初五/十五 |
| 坎 | 6 | 6日/6月/初六/十六 |
| 艮 | 7 | 7日/7月/初七/十七 |
| 坤 | 8 | 8日/8月/初八/十八 |

### 5.4 卦氣旺時應期

```
體卦旺相之時 = 應期

例：體卦為震（木）
- 春季（木旺）→ 應期在春季
- 或寅卯日（木日）→ 應期在寅卯日
```

## 七、分類占斷要點

### 7.1 求財

```
關鍵：看體用生克

吉象：
- 用生體 = 財來找我 = 大吉
- 體克用 = 我能得財 = 吉但費力
- 比和 = 平穩獲利

凶象：
- 體生用 = 財被耗損 = 損財
- 用克體 = 財難入手 = 不得

旺衰調整：
- 體卦旺相 = 有財運
- 體卦囚死 = 財運差
```

### 7.2 婚姻感情

```
關鍵：看體用是否和諧

吉卦：咸、恆、泰、既濟
凶卦：睽、否、未濟、離

判斷：
- 用生體 = 對方主動、有助益
- 體生用 = 我方付出多
- 比和 = 門當戶對
- 相克 = 有矛盾

互卦：看感情發展過程
變卦：看最終結果
```

### 7.3 事業工作

```
關鍵：體為自己，用為環境/上司

吉象：
- 用生體 = 得到支持
- 體旺 = 自身條件好

看進退：
- 乾、震、升、晉 = 宜進取
- 遯、蹇、困、剝 = 宜退守
```

### 7.4 健康疾病

```
關鍵：體為病人，用為病症

判斷：
- 體旺 = 體質好，病易癒
- 體衰 = 體質弱，需注意
- 用克體 = 病情重
- 體克用 = 能戰勝疾病

五行與臟腑：
- 木 = 肝膽
- 火 = 心
- 土 = 脾胃
- 金 = 肺
- 水 = 腎

哪個五行受克，對應臟腑有問題
```

### 7.5 出行

```
關鍵：看路途是否順利

吉卦：旅、漸、升、泰
凶卦：蹇、困、坎、屯

方向：生體之方為吉
時間：體卦旺時出行為佳
```

## 八、外應觀察

### 8.1 什麼是外應

```
外應 = 起卦時環境中的信號

古人相信：起卦的那一刻，天地人處於同步狀態，
環境中的一切都可能是「象」的顯現。
```

### 8.2 常見外應

| 外應 | 吉凶 | 說明 |
|------|------|------|
| 聽到喜訊 | 吉 | 事情順利 |
| 聽到哭聲 | 凶 | 有阻礙 |
| 見花開 | 吉 | 事情開展 |
| 見花落 | 凶 | 事情衰敗 |
| 見鳥飛 | 動 | 事情有變化 |
| 風起 | 巽 | 有信息、變動 |
| 雷響 | 震 | 驚動、開始 |
| 下雨 | 坎 | 險阻、滋潤 |

### 8.3 外應的使用原則

```
1. 自然發生：不是刻意尋找的
2. 即時性：起卦那一刻發生的
3. 顯著性：明顯引起注意的
4. 相關性：與問題有象徵關聯的

注意：外應是輔助判斷，不是主要依據
```

## 九、實戰案例

### 案例一：求財

**背景**：2026年春天，某人問能否獲得一筆投資回報

**起卦**：地火明夷，四爻動，變卦地水師

**分析**：
```
本卦：地火明夷（坤上離下）
動爻：四爻（在坤卦）
體：離（火）
用：坤（土）

體用關係：體（火）生用（土）= 體生用 = 耗損

季節：春天
- 離火在春季為「相」（木生火）
- 坤土在春季為「死」（木克土）

判斷：
1. 體生用 = 財被耗損，不利求財
2. 但體卦在春天有力（相位）
3. 變卦地水師 = 需要謹慎經營

結論：投資可能有損耗，不建議貿然投入。若要投資，需謹慎控制規模。
```

### 案例二：感情

**背景**：問與某人的感情前景

**起卦**：澤山咸，二爻動，變卦澤地萃

**分析**：
```
本卦：澤山咸（兌上艮下）
動爻：二爻（在艮卦）
體：兌（金）
用：艮（土）

體用關係：用（土）生體（金）= 用生體 = 大吉

咸卦本義：感應、交感，是婚姻吉卦

變卦萃：聚集，表示會走到一起

判斷：
1. 用生體 = 對方對我有助益
2. 咸卦 = 心意相通
3. 變卦萃 = 最終聚合

結論：感情發展順利，對方對我有好感，有望成功。
```

### 案例三：失物

**背景**：丟失錢包，問在何處

**起卦**：風地觀，五爻動，變卦山地剝

**分析**：
```
本卦：風地觀（巽上坤下）
動爻：五爻（在巽卦）
體：坤（土）
用：巽（木）

用卦巽：東南方

五行特徵：
- 巽為木，為風，為入
- 物品可能被風吹動，或在縫隙中

方位：東南方

進一步判斷：
- 巽為長女、為木
- 可能在女性長輩處，或木製家具附近

結論：錢包在東南方向，可能在木製家具的縫隙中，或與女性家人有關。
```

## 十、學習建議

### 10.1 入門階段（1-3個月）

```
1. 熟記八卦基本象義（背誦八卦歌）
2. 理解五行生克（畫圖理解）
3. 學會一種起卦方法（推薦時間起卦）
4. 練習分體用
5. 每天起一卦，記錄並驗證
```

### 10.2 進階階段（3-12個月）

```
1. 學習多種起卦方法
2. 深入理解64卦卦義
3. 學習應期推斷
4. 理解卦氣旺衰
5. 開始觀察外應
6. 累積案例，建立經驗庫
```

### 10.3 精通階段（1年以上）

```
1. 融會貫通各種判斷方法
2. 形成自己的解卦風格
3. 能快速直覺判斷
4. 結合修煉體驗深化理解
5. 不執著於技術，重視心態
```

### 10.4 重要提醒

```
1. 不迷信：占卦是參考，不是絕對
2. 不濫占：同一事不重複占卦
3. 重誠意：心誠則靈
4. 勤記錄：每卦都記錄驗證
5. 多反思：錯了要分析原因
6. 修心性：心清淨則卦準確
```
