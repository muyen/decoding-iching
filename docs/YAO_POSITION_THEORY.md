# 爻位系統理論：為什麼每爻意義不同

## 核心原理

每條爻辭的意義由以下因素決定：

```
爻義 = f(位置, 陰陽, 得位, 中正, 應, 承乘, 卦之時義)
```

---

## 一、六爻位置的固有意義

| 爻位 | 符號 | 階段 | 比喻 | 通則 |
|------|------|------|------|------|
| 初爻 | 初 | 萌芽 | 平民、開始 | 位卑力微，宜潛藏 |
| 二爻 | 二 | 發展 | 臣下、內部 | 內卦之中，得中則吉 |
| 三爻 | 三 | 轉折 | 過渡、危險 | 內卦極點，多凶多懼 |
| 四爻 | 四 | 躍升 | 近臣、外部 | 近君之位，多憂慮 |
| 五爻 | 五 | 高峰 | 君主、核心 | 九五之尊，最佳位 |
| 上爻 | 上 | 終結 | 退休、極端 | 事之終，物極必反 |

### 位置公式

```
吉凶傾向 = 基礎值 + 修正值

基礎值：
- 初爻：0（中性，看條件）
- 二爻：+1（內中，吉）
- 三爻：-1（內極，多凶）
- 四爻：0（近君，看條件）
- 五爻：+2（尊位，大吉）
- 上爻：-1（過極，多悔）
```

---

## 二、陰陽與得位

### 陽位與陰位

```
陽位（奇數）：初、三、五
陰位（偶數）：二、四、上
```

### 得位判斷

| 情況 | 組合 | 意義 | 效果 |
|------|------|------|------|
| 陽得位 | 九在初、三、五 | 剛居剛位 | +1 |
| 陰得位 | 六在二、四、上 | 柔居柔位 | +1 |
| 陽失位 | 九在二、四、上 | 剛居柔位 | 0或-1 |
| 陰失位 | 六在初、三、五 | 柔居剛位 | 0或-1 |

### 例外規則

**二、五位特殊**：
- 九二：剛失位但得中 → 吉
- 六五：柔失位但得中且為尊 → 往往大吉（如大有六五）

---

## 三、中正之道

### 「中」的力量

```
中位 = 二爻（內卦之中）、五爻（外卦之中）
得中 = 吉的基礎
```

### 「中正」= 中 + 得位

| 爻位 | 中 | 正 | 中正 | 判斷 |
|------|-----|-----|------|------|
| 九二 | ✓ | ✗ | 半 | 剛中，吉 |
| 六二 | ✓ | ✓ | ✓ | 中正，大吉 |
| 九五 | ✓ | ✓ | ✓ | 中正，大吉 |
| 六五 | ✓ | ✗ | 半 | 柔中，多吉 |

---

## 四、應與不應

### 應的配對

```
初爻 ←→ 四爻
二爻 ←→ 五爻
三爻 ←→ 上爻
```

### 應的條件

| 情況 | 效果 | 說明 |
|------|------|------|
| 陰陽相應 | +1 | 一陰一陽才能相應 |
| 同性無應 | 0或-1 | 兩陽或兩陰無法相應 |
| 二五相應 | 特別吉 | 臣君相得 |

### 例子

- 乾卦：六爻皆陽，所有應位都是同性 → 無應
- 屯卦：六二(陰)與九五(陽) → 有應，但「屯其膏」表示難以施澤

---

## 五、承與乘

### 定義

```
承 = 下爻對上爻的關係（下承上）
乘 = 上爻對下爻的關係（上乘下）
```

### 承乘法則

| 關係 | 組合 | 意義 | 效果 |
|------|------|------|------|
| 陰承陽 | 陰爻在陽爻之下 | 柔順承剛 | 吉 |
| 陽承陰 | 陽爻在陰爻之下 | 不順 | 凶 |
| 陰乘陽 | 陰爻在陽爻之上 | 柔凌剛 | 不吉 |
| 陽乘陰 | 陽爻在陰爻之上 | 剛制柔 | 正常 |

### 例子

- 屯卦六二「乘馬班如」：六二(陰)乘初九(陽) → 陰乘陽，故「班如」(徘徊)
- 師卦九二：唯一陽爻被眾陰承載 → 陰承陽，故「吉」

---

## 六、卦之時義

每卦有其「時」，同一位置在不同卦中意義不同：

### 時義修正

```
爻義 = 位置義 × 時義係數

時義係數：
- 順時（爻義符合卦旨）：+1
- 逆時（爻義違背卦旨）：-1
```

### 例子

| 卦 | 時義 | 三爻通常凶 | 實際 |
|----|------|-----------|------|
| 謙 | 謙虛 | 九三：勞謙 | 吉（謙之時義救了危位）|
| 履 | 謹慎 | 六三：履虎尾 | 凶（危位加危行）|

---

## 七、綜合公式

```
爻義 = 位置基礎值
     + 得位修正(+1/0/-1)
     + 得中修正(+1/0)
     + 相應修正(+1/0/-1)
     + 承乘修正(+1/0/-1)
     × 時義係數(+1/-1)
```

### 示例推導

**乾卦九五「飛龍在天，利見大人」**：
```
位置：五（+2，尊位）
得位：九在五（陽在陽位，+1）
得中：五為外卦中（+1）
相應：與二爻（九二），同性無應（0）
承乘：乘九四（陽乘陽，中性）
時義：乾之時為「剛健」，五爻正是剛健顯現（×1）

總分 = 2+1+1+0+0 = 4（大吉）
```

**履卦六三「履虎尾，咥人，凶」**：
```
位置：三（-1，危位）
得位：六在三（陰在陽位，-1）
得中：否（0）
相應：與上九，陰陽相應（+1）
承乘：乘九二（陰乘陽，-1）
時義：履之時為「謹慎」，六三冒進（×-1，加重凶）

總分 = -1-1+0+1-1 = -2，再因時義加重 → 凶
```

## 八、特殊爻的解讀

### 用九、用六

只有乾坤有「用九」「用六」：
- **用九**：「見群龍無首，吉」→ 純陽之極，宜去中心化
- **用六**：「利永貞」→ 純陰之極，宜恆守

### 動爻（變爻）

```
動爻 = 老陽(9)或老陰(6)
老陽 → 變陰
老陰 → 變陽

動爻是變化的關鍵點，其爻辭特別重要
```

## 九、爻位與人事對應

| 爻位 | 人事 | 時間 | 空間 | 身體 |
|------|------|------|------|------|
| 初 | 平民、新手 | 開始 | 地下、基層 | 足 |
| 二 | 臣下、幹部 | 早期 | 內部 | 腿、腹 |
| 三 | 過渡者 | 轉折 | 邊界 | 腰 |
| 四 | 近臣、高管 | 中後期 | 外部 | 胸 |
| 五 | 君主、領導 | 高峰 | 核心 | 頸、面 |
| 上 | 退休者、極端 | 結束 | 最高/最遠 | 頭 |

## 十、實用判斷流程

```
1. 確定動爻位置
2. 查位置基礎吉凶（初中、二吉、三凶、四懼、五吉、上悔）
3. 判斷得位與否
4. 判斷是否得中
5. 判斷與應位的關係
6. 判斷與相鄰爻的承乘
7. 結合卦之時義綜合判斷
```
