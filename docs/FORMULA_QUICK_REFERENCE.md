## 速查規則

### 看到這些 → 判吉

```
✓ 五位 + 任何吉詞
✓ 坤+震 或 坤+巽 組合
✓ 「元吉」「大吉」「終吉」「无不利」
✓ 「貞吉」且沒有凶詞
```

### 看到這些 → 判凶

```
✗ 三位 + 陽爻 + 沒有吉詞
✗ 艮+艮 或 坎+坎 組合
✗ 「凶」「災」且沒有吉詞
✗ 「貞凶」
```

### 看到這些 → 判中

```
○ 「吝」(100%是中)
○ 「无咎」且沒有其他吉詞
○ 「婦吉夫凶」「小吉大凶」等混合句
```

## 預測流程

```
1. 先看爻辭有沒有條件句 (征凶居吉、小吉大凶等)
   → 有的話直接判定

2. 再看有沒有高可信關鍵詞 (元吉、凶、災等)
   → 有的話直接判定

3. 最後查結構表
   → LOOKUP[下卦, 上卦, 爻位] 得到傾向
```
