**創建日期**：2026-01-20
**項目目標**：為384條爻辭提供基於數據分析和結構研究的深度解釋

## 項目背景

傳統的爻辭解釋往往只是「帶過古書」，缺乏新意和深度洞見。本項目通過以下方法創建具有突破性的解釋：

1. **量化分析**：套用爻義公式計算每爻的吉凶分數
2. **四維關係**：分析錯卦、綜卦、互卦、變卦的影響
3. **統計驗證**：對照詞頻統計和模式分析驗證解讀
4. **新發現**：提出傳統古書未曾明言的創新見解

## 核心發現摘要

### 挑戰傳統觀點

| 傳統觀點 | 本研究發現 |
|----------|------------|
| 守正必吉 | 「貞凶」出現8次，守正需要時機 |
| 四爻多懼必凶 | 四爻「无咎」最多(22.2%)，是補救之位 |
| 上爻物極必反 | 上爻「悔亡」8次，「終吉」7次，有選擇 |
| 當位最重要 | 當位影響僅佔3%，時義佔40% |
| 利見大人普適 | 只出現在五爻和卦辭 |

### 吉凶因素權重

```
時義(40%) > 爻位(25%) > 得中(15%) > 應(10%) > 承乘(7%) > 當位(3%)
```

### 核心公式

```
爻義 = [(位置基礎值 + 陰陽修正 + 得位修正 + 得中修正 + 應修正 + 承乘修正)
        × 時義係數 × 卦序係數]
      + 互卦影響 + 錯綜卦語境
```

## 如何使用本解釋

### 步驟一：確定卦象
根據占卜或研究需要，確定要查閱的卦和爻位。

### 步驟二：查看時義分類
每卦開頭標明其時義類型（順時/逆時/轉換/等待），這是判斷吉凶的首要因素。

### 步驟三：查看公式計算
每爻都有完整的公式計算過程，可以理解吉凶判斷的依據。

### 步驟四：關注新發現
每爻解釋包含一個「新發現」部分，提供傳統解讀未提及的洞見。

## 特殊標記說明

在爻辭解釋中，使用以下標記：

- **【逆轉】**：此爻的吉凶與位置基礎預期相反
- **【符合】**：此爻完全符合統計模式
- **【例外】**：此爻是統計模式的例外案例
- **【新發現】**：此解讀為本研究首次提出
