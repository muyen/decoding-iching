# 64卦系統推導

基於 SYSTEM_THEORY.md 中的生成規則，系統性推導所有64卦的意義。

---

## 推導公式

```
卦義 = 上卦功能(作用於/影響)下卦功能 + 位置關係判定
```

- 上卦：乾(君、健)
