# 易經系統理論：逆向工程

## 目標

從說卦傳等原典中，提取易經系統的**生成規則**，而非僅僅記錄「是什麼」。

目的：
1. 理解系統設計原則
2. 能夠從結構**推導**意義
3. 驗證推導結果與傳統解釋的一致性
4. 用於A/B測試，評估解釋的預測準確性

## 第一層：二進制與家族模型

### 1.1 基本結構

```
陰爻 ⚋ = 0 = 斷開、接收、被動
陽爻 ⚊ = 1 = 連續、發出、主動
```

三爻組合 → 8種狀態 → 八卦

### 1.2 家族生成模型（說卦傳第十章）

> 「乾天也，故稱父；坤地也，故稱母；
> 震一索而得男，故謂之長男；巽一索而得女，故謂之長女；
> 坎再索而得男，故謂之中男；離再索而得女，故謂之中女；
> 艮三索而得男，故謂之少男；兌三索而得女，故謂之少女。」

## 第四層：重卦意義生成規則

### 4.1 基本公式

```
重卦意義 = f(上卦, 下卦, 位置關係, 互卦)
```

其中：
- **上卦**：外在、結果、環境、對方、未來
- **下卦**：內在、起因、自身、主體、過去
- **位置關係**：自然 vs 反常
- **互卦**：內在本質、過程

### 4.2 位置關係判定

**自然位置**（符合常理）：
- 天(乾)在上、地(坤)在下 → 正常但不交流
- 水(坎)在下、火(離)在上 → 正常烹飪位置
- 山(艮)在上 → 穩定
- 澤(兌)在下 → 蓄積

**反常位置**（違反常理）：
- 地在上、天在下 → 交流（泰）
- 火在下、水在上 → 未完成（未濟）
- 山在下、澤在上 → 損下益上（損）

### 4.3 功能組合規則

```
上卦功能 + 下卦功能 = 情境描述

例：
風(散) + 火(明) = 風火家人
→ 風散佈火的光明 → 家庭教化傳播

山(止) + 水(陷) = 山水蒙
→ 山下有險水 → 蒙昧、需要啟發
```

## 第五層：驗證方法

### 5.1 內部一致性

1. 推導的意義是否與卦名相符？
2. 配對卦（文王序）是否呈現互補關係？
3. 互卦是否反映內在本質？
4. 錯卦（陰陽互換）是否呈現對立關係？

### 5.2 與傳統文獻對比

對比來源：
- 卦辭、爻辭原文
- 彖傳解釋
- 象傳解釋
- 歷代註釋

評分標準：
- 完全一致：意義相同
- 部分一致：核心相同，細節不同
- 不一致：需要重新檢視推導

### 5.3 預測驗證（A/B測試）

使用梅花易數實驗框架：
- A組：傳統解釋
- B組：系統推導解釋
- 測量：預測準確率

## 第六層：變卦機制 — 動態變化理論

### 6.1 什麼是變卦？

易經不只描述靜態狀態，更重要的是**變化過程**。

```
本卦 → 變卦
（現狀）  （趨勢/結果）
```

**關鍵概念**：
- **本卦**（主卦）：當前的狀態和情境
- **變卦**（之卦）：事物發展的方向和結果
- **動爻**：引發變化的關鍵因素

### 6.2 變爻的數學原理

爻有四種狀態（四象）：

| 狀態 | 數值 | 性質 | 是否變化 |
|------|------|------|----------|
| 老陽 | 9 | ⚊ 陽極 | ✅ 變陰 |
| 少陽 | 7 | ⚊ 陽 | ❌ 不變 |
| 少陰 | 8 | ⚋ 陰 | ❌ 不變 |
| 老陰 | 6 | ⚋ 陰極 | ✅ 變陽 |

**為什麼會變？**

這基於**物極必反**的哲學原理：
```
陽極（老陽9）→ 物極必反 → 變陰
陰極（老陰6）→ 否極泰來 → 變陽
```

「少」者尚未成熟，故穩定；「老」者已至極點，故轉化。

### 6.3 變卦的解讀邏輯

**解讀公式**：
```
完整解讀 = 本卦（情境）+ 動爻（關鍵）+ 變卦（結果）
```

**詳細說明**：

| 成分 | 代表意義 | 解讀方式 |
|------|----------|----------|
| 本卦 | 事情的基本性質 | 用上下卦功能組合解讀 |
| 動爻 | 變化的引爆點 | 該爻位置決定變化發生在哪個層面 |
| 變卦 | 最終走向 | 用上下卦功能組合解讀 |
| 互卦 | 過程中的內在狀態 | 取2-4爻為下，3-5爻為上 |

### 6.4 動爻位置的意義

六爻從下到上代表事物發展的六個階段：

| 爻位 | 階段 | 意義 | 動爻在此位的解讀 |
|------|------|------|------------------|
| 初爻 | 開始 | 事物萌芽，潛藏 | 變化發生在起始階段 |
| 二爻 | 發展 | 居中得位，顯現 | 變化發生在發展過程 |
| 三爻 | 過渡 | 下卦之極，轉折 | 內部因素導致轉變 |
| 四爻 | 進入 | 上卦之始，接近 | 外部因素開始介入 |
| 五爻 | 成熟 | 君位，主導 | 核心決策導致變化 |
| 上爻 | 終結 | 事物之極，窮盡 | 物極必反，根本轉變 |

### 6.5 多爻變動的處理

| 變爻數量 | 解讀方法 |
|----------|----------|
| 0爻變 | 以本卦卦辭為主 |
| 1爻變 | 以動爻爻辭為主 |
| 2爻變 | 以兩動爻爻辭為主（下爻為因，上爻為果）|
| 3爻變 | 本卦、變卦並重 |
| 4爻變 | 以變卦兩靜爻為主 |
| 5爻變 | 以變卦一靜爻為主 |
| 6爻變 | 乾坤用用九用六，餘卦用變卦 |

### 6.6 變卦與本卦的關係類型

```
設 本卦 = A，變卦 = B

關係判斷：
1. A和B是否為錯卦（陰陽全反）→ 根本性質轉變
2. A和B是否為綜卦（上下顛倒）→ 視角/立場轉變
3. A和B僅差一爻 → 微調性變化
4. A和B差多爻 → 劇烈變化
```

## 第七層：語義來源 — 為什麼這樣命名？

### 7.1 卦名的構詞邏輯

易經的卦名不是隨意取的，而是遵循**以象取義**的原則。

**核心公式**：
```
卦名 = 最能代表「上卦作用於下卦」這一情境的漢字
```

### 7.2 八卦基本象的語源

| 卦 | 字源分析 | 為什麼這樣稱呼 |
|----|----------|----------------|
| **乾** | 甲骨文像太陽升起 | 天體運行不息，「健」的極致 |
| **坤** | 從土從申，大地延展 | 承載萬物，「順」的極致 |
| **震** | 雨+辰，雷雨之象 | 雷聲震動萬物，故為「動」 |
| **巽** | 兩個「巳」，蛇入穴 | 風無孔不入，故為「入」 |
| **坎** | 欠+土，地面凹陷 | 水往低處流，陷入坑中 |
| **離** | 鳥網之象，附著 | 火附著於物才能燃燒，故為「麗」|
| **艮** | 眼睛回望之象 | 回頭看=停下來，故為「止」 |
| **兌** | 人張口說話之象 | 張口表達喜悅，故為「說（悅）」|

### 7.3 重卦命名的系統規則

**規則1：取象命名**
```
觀察上下卦組合呈現的自然現象 → 取名

例：
水天需 = 水在天上 = 雲 = 等待降雨 → 需（等待）
火地晉 = 火（太陽）出地面 = 日出 → 晉（上升）
```

**規則2：取義命名**
```
分析上下卦功能的互動關係 → 取名

例：
地天泰 = 地氣下降 + 天氣上升 = 交流 → 泰（通暢）
天地否 = 天氣上升 + 地氣下降 = 背離 → 否（閉塞）
```

**規則3：取德命名**
```
從組合中提取人事意義 → 取名

例：
天澤履 = 剛健(天)遇喜悅(澤) = 以和悅態度侍奉尊者 → 履（禮）
地山謙 = 大地中有高山 = 內高外低 → 謙（謙虛）
```

### 7.4 64卦命名邏輯總表

| 類型 | 命名邏輯 | 代表卦例 |
|------|----------|----------|
| **純卦(8個)** | 重複強調本性 | 乾(純剛)、坤(純柔)、震(純動)... |
| **交泰類** | 上下交流與否 | 泰(交)、否(塞)、既濟(成)、未濟(未成) |
| **自然象類** | 直接取自然現象 | 雷(震)、風(巽)、火(離)、水(坎) |
| **人事類** | 取人事情境 | 師(軍隊)、比(親近)、家人、歸妹 |
| **動態類** | 描述變化過程 | 剝(剝落)、復(復返)、損、益 |
| **時機類** | 強調時間要素 | 需(等待)、漸(漸進)、晉(上升) |

### 7.5 記憶口訣

**八卦功能口訣**（說卦傳第四章）：
```

艮止兌說乾君坤藏
```

## 第八層：系統總結 — 大道至簡

### 8.1 易經系統的數學本質

```
易經 = 二進制系統(陰陽) × 組合數學(八卦×八卦) × 動態演化(變爻)
```

**具體展開**：
- 2¹ = 2 → 陰陽兩儀
- 2² = 4 → 四象（老陽、少陽、少陰、老陰）
- 2³ = 8 → 八卦
- 8² = 64 → 六十四卦
- 6爻 × 4態 = 4⁶ = 4096 → 所有可能的占卦組合

### 8.2 意義生成的統一公式

```
卦義 = 上卦功能(作用於)下卦功能
     × 交流係數
     × 陰陽消長係數
     × 時位係數

其中：
- 上卦功能、下卦功能：來自說卦傳的八個動詞
- 交流係數：基於能量方向判定（+1通、-1塞、0中性）
- 陰陽消長係數：基於陰陽爻數量比例
- 時位係數：基於動爻位置
```

### 8.3 為什麼這個系統有效？

**假說**：易經是古人對**模式識別**的編碼系統

1. **二元對立**：世間萬事可分陰陽（是/否、進/退、明/暗...）
2. **組合窮盡**：64種基本情境覆蓋大多數人生場景
3. **動態演化**：變爻機制模擬事物發展過程
4. **類比推理**：用自然象徵對應人事（如水險→人生危機）

**這不是神秘主義，而是**：
```
模式識別 + 類比推理 + 概率思維 = 決策輔助工具
```
