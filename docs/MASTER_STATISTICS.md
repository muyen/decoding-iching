## 12. 核心結論

### 12.1 確認的傳統智慧

1. **"九五之尊"** ✓ - 位置5確實最佳（48.4%吉率，6.2%凶率）
2. **"三多凶"** ✓ - 位置3確實最危險（10.9%吉率，32.8%凶率）
3. **"得中"** ✓ - 中位（2,5爻）明顯優於非中位
4. **"三五同功"** ✓ - 爻3和爻5有最強正相關（r=+0.41）

### 12.2 需要修正的認知

1. ~~"位置1吉率最高"~~ - 錯誤！位置5才是最高
2. ~~"謙卦六爻皆吉"~~ - 錯誤！謙卦是5吉0凶（83.3%），不是100%
3. ~~"當位比不當位好"~~ - 差異極小，不是決定性因素
4. ~~"艮在下最好"~~ - 錯誤！艮在下淨分較低

### 12.3 新發現

1. **外卦優於內卦** - 4-6爻整體優於1-3爻
2. **錯卦正相關** - 錯卦整體呈正相關（r=+0.46），但經典對立卦符合傳統
3. **動爻為凶更可能改善** - 變卦後平均+1.11
4. **上下卦不同優於相同** - 異卦組合平均+1.21 vs 同卦+0.12
5. **六五>九五** - 陰在五位吉率56.2%，陽在五位40.6%，柔中帶剛更優
6. **否極泰來** - 爻4凶時，爻5有60%機率是吉，0%機率凶
7. **乾卦全中** - 純陽乾卦0吉0凶，反而中性；坤卦2吉0凶
8. **艮卦爻3最差** - 0%吉率，50%凶率
9. **震卦爻5最佳** - 75%吉率，震整體差但爻5極好
