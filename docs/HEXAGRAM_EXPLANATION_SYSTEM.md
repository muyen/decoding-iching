# 卦象解釋系統 - 基於交互屬性

## 核心原理

```
卦象吉凶 = 下卦德性 × 上卦德性 × 交互效應 × 爻位修正
```

**不用五行，用卦德交互！**

## 八卦德性表 (說卦傳原文)

### 數字代表什麼？

```
數字 = 統計出來的「吉凶傾向分數」

+ (正數) = 讓整體更偏向「吉」
- (負數) = 讓整體更偏向「凶」
數字越大 = 影響力越強

白話說:
  +0.375 = 很加分，放這裡容易吉
  +0.229 = 有加分，稍微偏吉
  +0.062 = 幾乎沒影響
  -0.021 = 稍微扣分，略偏凶
```

## 交互效應規則 (白話版)

### 規則1: 穩定+衝勁 = 最佳 ★★

```
坤(順) + 震(動) = +0.83

白話:
  下面 = 穩定的地基，承載一切
  上面 = 往上衝的動力

  就像種子在土裡發芽往上長
  有穩定基礎 + 有向上動力 = 最好的配置

比喻: 公司有穩定後勤 + 前線積極進攻 = 成功
一句話: 「下面給力撐住，上面放心往前衝」
```

### 規則2: 停+停 = 最差 ✗✗

```
艮(止) + 艮(止) = -0.50

白話:
  下面在停，上面也在停
  兩個都不動 = 完全卡住

比喻: 車子熄火 + 前面塞車 = 哪都去不了
一句話: 「下面卡住，上面也卡住，動彈不得」
```

### 規則3: 危險+危險 = 更慘 ✗

```
坎(陷) + 坎(陷) = -0.33

白話:
  下面是坑洞，上面也是坑洞
  一個坑跳到另一個坑 = 越陷越深

比喻: 欠債 + 借高利貸還債 = 雪上加霜
一句話: 「坑裡面還有坑」
```

### 規則4: 想穩+被晃 = 衝突 ✗✗

```
離(麗) + 震(動) = -0.50

白話:
  下面需要穩定依附 (離=想靠著、附著)
  上面一直在震動 (震=動、變化)
  想穩定但一直被搖晃 = 站不穩

比喻: 想好好寫字 + 桌子一直在震 = 寫不好
一句話: 「下面想靠著休息，上面一直在搖」
```

### 規則5: 配合+配合 = 順利 ★

```
坤(順) + 坤(順) = +0.33

白話:
  下面很配合，上面也很配合
  大家都好講話 = 事情順利
  這是唯一可以疊加的正面特性

比喻: 團隊每個人都好溝通 = 合作愉快
一句話: 「上下都配合，啥事都好辦」
```

```

■ 交互
  順 + 動 → 強烈增益 (+0.57)
  原理: 大地柔順承載，雷動奮發其上
        萬物得順而動，生機勃發

■ 六爻: 吉吉吉吉吉中 (5吉0凶)

■ 總評: 大吉卦，總分+5

```

■ 交互
  止 + 止 → 負面疊加 (-0.52)
