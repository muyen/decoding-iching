# 第十九章：投資思維——風險與時機

---

## 投資的本質問題

投資的核心問題只有兩個：
1. **買什麼？**（選擇）
2. **什麼時候買/賣？**（時機）

易經對第二個問題特別有洞見。

---

## 市場週期的六爻分析

市場有周期，正好對應六爻：

```
位置    市場階段        特徵                  建議
───────────────────────────────────────────────────
1       築底期          恐慌、低迷            觀察、準備
2       復甦期          開始回升、懷疑        逐步建倉
3       加速期          快速上漲、興奮        謹慎、控制倉位
4       震盪期          高位震盪、分歧        減倉、觀望
5       瘋狂期          全民談股、新高        準備退出
6       泡沫破裂        暴跌、絕望            已經太晚
```

---

## 位置1：築底期——潛龍勿用

**市場特徵：**
- 連續下跌後的低迷
- 成交量萎縮
- 沒人談論股市
- 「股市已死」的聲音

**易經建議：** 觀察、準備，但不要急於抄底

```
✓ 正確做法：
- 準備資金
- 研究標的
- 等待確認信號
- 小倉位試探

✗ 錯誤做法：
- 「最低點」抄底（可能更低）
- 滿倉殺入
- 加槓桿
```

**數據提醒：** 位置1吉率26.7%，底部可能反覆。

---

## 位置2：復甦期——得中之位

**市場特徵：**
- 開始回升
- 但很多人還在懷疑
- 成交量逐步放大
- 「這是反彈還是反轉？」

**這是最好的建倉時機！**

```
✓ 正確做法：
- 逐步建倉
- 分散投資
- 設定止損
- 有紀律地執行

✗ 錯誤做法：
- 等「確認」才買（會錯過大部分漲幅）
- 一次性滿倉
- 不設止損
```

**數據提醒：** 位置2吉率46.7%，是風險收益比最好的時期。

---

## 位置3：加速期——三多凶

**市場特徵：**
- 快速上漲
- 財經新聞增多
- 開始有人談論股票
- FOMO（害怕錯過）情緒

**這是最危險的階段！**

```
為什麼危險？

- 漲幅已經可觀，回調風險增加
- 追高的人開始進場
- 市場情緒從「謹慎」變「樂觀」
- 很多人在這個階段虧損（追高被套）
```

**易經建議：**

```
✓ 正確做法：
- 不追高
- 控制倉位
- 設定止盈
- 如果已經有倉位，持有但謹慎

✗ 錯誤做法：
- 看到漲就追
- 加槓桿
- 「再漲一點就賣」
- 忽視風險
```

**數據提醒：** 位置3吉率僅6.7%！追高是大多數人虧損的原因。

---

## 位置4：震盪期——近君之位

**市場特徵：**
- 高位震盪
- 多空分歧加大
- 有人說「還會漲」，有人說「已見頂」
- 成交量不規則

**易經建議：** 謹慎、降低倉位

```
✓ 正確做法：
- 減少倉位
- 提高現金比例
- 觀察市場信號
- 不要做重大決策

✗ 錯誤做法：
- 在震盪中頻繁操作
- 被洗出去又追回來
- 忽視警告信號
```

---

## 位置5：瘋狂期——九五之尊

**市場特徵：**
- 連創新高
- 全民談股
- 新股民大量進場
- 「這次不一樣」的聲音
- 極度樂觀

**這是應該退出的時機！**

```
雖然叫「九五之尊」，
但在投資中，位置5意味著：
- 市場已經「登頂」
- 能賺的錢已經賺得差不多了
- 風險遠大於收益
```

**易經建議：** 準備退出

```
✓ 正確做法：
- 逐步賣出
- 獲利了結
- 保住勝利果實
- 不要貪心

✗ 錯誤做法：
- 「再漲一點」
- 加倉追高
- 「這次不一樣」
- 忽視泡沫信號
```

**巴菲特的智慧：** 「在別人貪婪時恐懼」——這正是位置5。

---

## 位置6：泡沫破裂——亢龍有悔

**市場特徵：**
- 暴跌
- 恐慌
- 踩踏
- 「怎麼會這樣？」

**如果你在這裡，可能已經太晚了。**

```
教訓：
- 位置6的風險在位置5就應該識別
- 「物極必反」是確定的
- 不要試圖抓住最後一段漲幅
```

---

## 投資決策的「卦象框架」

### Step 1：判斷市場的「位置」

```
問自己：
- 市場現在是什麼情緒？（恐慌/謹慎/樂觀/瘋狂）
- 成交量如何？（萎縮/正常/放大/極端）
- 周圍的人在談論投資嗎？
- 新聞是正面還是負面？
```

### Step 2：判斷你的「下卦」

```
你的投資心態：
乾 = 想積極進攻
坤 = 想穩健保守
艮 = 想等待觀望
震 = 想快速行動
坎 = 感到恐懼
離 = 被熱點吸引
```

### Step 3：評估組合

```
市場（上卦）+ 你的心態（下卦）= 投資決策

例如：
- 市場瘋狂（乾）+ 你想追高（乾）= 乾乾（極端，危險）
- 市場低迷（坤）+ 你想抄底（震）= 復卦（可能是機會）
- 市場震盪（坎）+ 你感到恐懼（坎）= 坎坎（純險，謹慎）
```

## 風險管理的「坎卦智慧」

坎卦代表「險」，但也代表「智慧」。

### 坎的正面意義

```
水的特性：
- 遇到阻礙會繞過
- 最終會找到出路
- 柔軟但持久
- 向下流動（順勢）
```

### 應用到風險管理

```
1. 接受風險的存在（不逃避）
2. 學會「繞過」而不是硬碰（止損）
3. 保持流動性（不把所有錢鎖死）
4. 順勢而為（不和市場對著幹）
```

---

## 「止」的智慧：艮卦在投資中

艮卦代表「止」，在投資中極其重要：

```
何時該「止」：

止損：
- 虧損達到預設比例
- 買入邏輯不再成立
- 發現自己錯了

止盈：
- 盈利達到預期
- 市場進入位置5-6
- 估值過高

止動：
- 市場不明朗
- 自己情緒不穩定
- 沒有好的機會
```

**艮下卦吉率58.3%——穩固的底線是投資成功的基礎。**
