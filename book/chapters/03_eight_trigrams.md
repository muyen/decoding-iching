# 第三章：八卦——八個基本字母

---

## 從字母到單詞

英文有26個字母，組合成數萬個單詞。
中文有數千個基本字，組合成無數詞語。

易經有8個「字母」——**八卦**。

這8個字母組合成64個「單詞」——**六十四卦**。

理解了這8個字母，你就掌握了易經的基礎詞彙。

---

## 八卦是怎麼來的？

把3個陰陽爻疊在一起，會產生多少種組合？

```
2 × 2 × 2 = 8 種
```

這就是八卦：

```
卦象    名稱    二進制    數值    自然象徵
━━━
☰     乾      111       7      天
☱     兌      110       6      澤
☲     離      101       5      火
☳     震      100       4      雷
☴     巽      011       3      風
☵     坎      010       2      水
☶     艮      001       1      山
☷     坤      000       0      地
```

## 八個字母的性格

每個卦都有獨特的「性格」，就像字母有自己的發音和感覺。

### ☰ 乾（111）— 天

**關鍵詞：** 剛健、創造、領導

乾是純陽，代表：
- 天的無限高遠
- 創造的力量
- 領導者的特質

**現代類比：**
- 創業者的開創精神
- CEO的決斷力
- 創新者的突破

**注意：** 純陽也意味著過剛易折。乾卦上爻說「亢龍有悔」——飛得太高會後悔。

---

### ☷ 坤（000）— 地

**關鍵詞：** 順從、包容、承載

坤是純陰，代表：
- 大地的承載力
- 順從配合的智慧
- 厚德載物

**現代類比：**
- 優秀的執行者
- 支持型的團隊成員
- 穩健的基礎建設

**我們的發現：** 坤作為上卦時，吉率高達44.4%——在所有八卦中排名第一！

為什麼？因為「順從在上」是最佳的外在姿態。謙虛、柔順的外表，往往帶來好結果。

---

### ☶ 艮（001）— 山

**關鍵詞：** 停止、穩重、界限

艮代表山，意味著：
- 知道何時停止
- 穩如泰山的定力
- 清晰的界限

**現代類比：**
- 知道說「不」的人
- 有底線的談判者
- 懂得止損的投資者

**我們的發現：** 艮作為下卦時，吉率高達58.3%——在所有八卦中排名第一！

為什麼？因為「止為基」——穩固的內在基礎是成功的根本。

**這解釋了謙卦（坤上艮下）為什麼六爻皆吉：** 它同時擁有最佳上卦（坤）和最佳下卦（艮）！

---

### ☱ 兌（110）— 澤

**關鍵詞：** 愉悅、交流、柔和

兌代表湖澤，意味著：
- 喜悅和樂
- 言語溝通
- 柔和親切

**現代類比：**
- 社交達人
- 公關專家
- 談判高手

**我們的發現：** 兌作為上卦時，吉率只有8.3%——在所有八卦中排名最後！

為什麼？因為「悅在上」意味著輕浮。過度追求表面的愉悅和討好，往往帶來問題。

---

### ☲ 離（101）— 火

**關鍵詞：** 光明、依附、美麗

離代表火，意味著：
- 光明照耀
- 需要依附（火需要燃料）
- 文明美麗

**現代類比：**
- 品牌形象
- 公眾關注
- 藝術創作

---

### ☵ 坎（010）— 水

**關鍵詞：** 險阻、智慧、流動

坎代表水，意味著：
- 危險與挑戰
- 深邃的智慧
- 靈活變通

**現代類比：**
- 危機管理
- 深度思考
- 適應變化

---

### ☳ 震（100）— 雷

**關鍵詞：** 行動、震動、突破

震代表雷，意味著：
- 突然的行動
- 震撼性的開始
- 衝破阻礙

**現代類比：**
- 產品發布
- 市場突破
- 創新行動

---

### ☴ 巽（011）— 風

**關鍵詞：** 滲透、順從、漸進

巽代表風，意味著：
- 無孔不入的滲透
- 順勢而為
- 漸進式改變

**現代類比：**
- 市場滲透策略
- 軟性影響力
- 循序漸進的改革

## 記憶技巧

八卦看起來複雜，其實有規律：

### 按陽爻數量分組

```
3個陽爻：乾（111）— 純陽
2個陽爻：兌（110）離（101）震（100）
1個陽爻：巽（011）坎（010）艮（001）
0個陽爻：坤（000）— 純陰
```

### 對照記憶

```
乾（111）↔ 坤（000）  天 vs 地
兌（110）↔ 艮（001）  澤 vs 山
離（101）↔ 坎（010）  火 vs 水
震（100）↔ 巽（011）  雷 vs 風
```

每一對都是互補的：把所有0變1，1變0，就得到對應的卦。

## 商業應用：團隊角色

用八卦描述團隊中的角色：

```
角色           八卦  特點
───────────────────────────────
創始人/願景者  乾    創造方向
運營/執行者    坤    落實執行
財務/風控      艮    設定界限
銷售/公關      兌    對外交流
品牌/設計      離    塑造形象
策略/分析      坎    深度思考
產品/創新      震    突破創新
市場/滲透      巽    漸進擴張
```
一個完整的團隊需要不同類型的人。過多的「乾」會衝突，過多的「坤」會缺乏方向。
