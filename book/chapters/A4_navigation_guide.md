## 什麼是「變爻」？怎麼用？

### 基本概念

每個卦有6個爻（線）。**變一個爻**，就會變成另一個卦。

```
例：乾卦變第4爻 → 履卦

乾卦          變第4爻        履卦
━━━ 第6爻     ━━━           ━━━
━━━ 第5爻     ━━━           ━━━
━━━ 第4爻  →  ━ ━ (陽變陰)   ━ ━
━━━ 第3爻     ━━━           ━━━
━━━ 第2爻     ━━━           ━━━
━━━ 第1爻     ━━━           ━━━
```

### 表格怎麼看

當你看到「4→履」，意思是：

| 寫法 | 意思 |
|:-----|:-----|
| 4→履 | 變第4爻，會變成履卦 |
| 6→臨 | 變第6爻，會變成臨卦 |
| 3→謙 | 變第3爻，會變成謙卦 |

### 實際應用：不是改卦象，是改行動

**重要**：你不是真的去「改變」卦象。卦象代表你的現狀。

「變爻」的意思是：**改變那個位置代表的事情**。

```
爻位代表的層面：
第1爻 = 起步、基礎、根本
第2爻 = 發展、穩定、內在
第3爻 = 轉折、決定、風險點
第4爻 = 適應、調整、外在開始
第5爻 = 主導、發揮、核心位置
第6爻 = 終結、轉型、結束階段
```

### 完整範例

**情境**：你占到「師卦」，查表發現是🔴排斥子，建議「走」，最佳變爻「6→臨」。

**解讀**：
1. 師卦 = 你目前在帶領團隊打仗的狀態，吉率只有17%
2. 建議「走」= 這個狀態不適合久留
3. 「6→臨」= 改變第6爻的層面（終結、轉型）
4. 臨卦 = 主動面對、進入的狀態，吉率83%

**行動建議**：
- 第6爻代表「結束階段」
- 所以你應該思考：如何結束目前的「打仗」狀態，轉向「主動面對」的狀態
- 具體可能是：結束對抗、結束消耗戰、轉向合作或主動出擊

### 簡單記憶

```
看到「X→某卦」：
1. X = 要改變的層面（1=根本, 2=發展, 3=決定, 4=調整, 5=主導, 6=轉型）
2. 某卦 = 改變後會進入的狀態
3. 行動 = 在那個層面做出改變
```
