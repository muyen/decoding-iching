乾卦和坤卦是易經的「父」和「母」——所有其他卦都由這兩者交合而生。

```

## 乾卦：純陽的力量

### 卦辭

> 「乾：元亨利貞。」
> 乾卦：大吉大利，利於守正。

四個字「元亨利貞」是易經最高的評價：
- **元**：大、始、首
- **亨**：通達、順利
- **利**：有利、吉祥
- **貞**：守正、堅持

### 乾卦六爻的完整旅程

乾卦的六爻講述了一條龍的一生：

```
上九：亢龍有悔     — 飛得太高，後悔
九五：飛龍在天     — 大展宏圖
九四：或躍在淵     — 準備起飛
九三：終日乾乾     — 努力奮鬥
九二：見龍在田     — 嶄露頭角
初九：潛龍勿用     — 潛伏等待
```

這是一個完整的英雄旅程：
- 從潛伏（初九）
- 到嶄露（九二）
- 到奮鬥（九三）
- 到躍升（九四）
- 到巔峰（九五）
- 到過度（上九）

### 乾卦的關鍵教訓

**初九：潛龍勿用**

```
時機未到，不要行動。
即使你有能力，也要等待。
```

**九三：君子終日乾乾，夕惕若厲，無咎**

```
整天努力，晚上還要保持警惕。
位置3是危險的，只有加倍謹慎才能「無咎」。
```

**九五：飛龍在天，利見大人**

```
這是最好的位置。
可以大展宏圖，適合見到重要的人。
「九五之尊」的來源。
```

**上九：亢龍有悔**

```
飛得太高會後悔。
這是位置6的警告：物極必反。
知道何時停止。
```

## 坤卦：純陰的智慧

### 卦辭

> 「坤：元亨，利牝馬之貞。」
> 坤卦：大吉通達，利於如母馬般守正。

坤卦也是「元亨」，但加了一個條件：「牝馬之貞」——像母馬一樣的柔順堅貞。

### 坤卦的核心教導

```
初六：履霜，堅冰至
      踩到霜，知道堅冰要來。
      → 見微知著，提前準備

六二：直方大，不習無不利
      正直方正廣大，不必刻意學習也無不利。
      → 坤的本質是自然而然

六三：含章可貞
      含蓄內斂的美德。
      → 不顯露，但有內涵

六四：括囊，無咎無譽
      紮緊口袋，沒有災禍也沒有榮譽。
      → 謹慎時期

六五：黃裳，元吉
      黃色的下裳，大吉。
      → 居中守正，大吉

上六：龍戰於野，其血玄黃
      龍在野外戰鬥，流血。
      → 陰到極端，會與陽爭鬥
```

### 坤卦的關鍵教訓

**「先迷後得主」**

```
坤卦說：「先迷後得主，利。」

先（如果先行）迷（會迷失）
後（如果跟隨）得主（會找到主人/方向）

這是坤的智慧：不要搶在前面，跟隨會更好。
```

**「厚德載物」**

```
坤是大地，能承載萬物。
不是因為軟弱，而是因為包容。
```

## 否泰：乾坤的兩種組合

當乾和坤組合時，順序不同，結果完全不同：

### 泰卦（坤上乾下）

```
     ━ ━  ┐
     ━ ━  │ 坤在上
     ━ ━  ┘
     ━━━  ┐
     ━━━  │ 乾在下
     ━━━  ┘

地天泰 = 通泰
```

為什麼「地在上、天在下」反而好？

```
天氣（陽）的特性：向上升
地氣（陰）的特性：向下降

在泰卦中：
- 天在下，其氣向上升
- 地在上，其氣向下降
- 兩者在中間交匯！
= 交流、通暢、和諧
```

### 否卦（乾上坤下）

```
     ━━━  ┐
     ━━━  │ 乾在上
     ━━━  ┘
     ━ ━  ┐
     ━ ━  │ 坤在下
     ━ ━  ┘

天地否 = 閉塞
```

為什麼「天在上、地在下」反而不好？

```
在否卦中：
- 天在上，其氣向上升（更往上）
- 地在下，其氣向下降（更往下）
- 兩者越離越遠！
= 不交流、閉塞、分離
```

### 深刻的啟示

```
「正確」的位置不一定好
「顛倒」的位置不一定壞

關鍵是：能不能交流、能不能互動
```

這個道理適用於：
- 上下級關係（能不能溝通）
- 夫妻關係（能不能交心）
- 國際關係（能不能合作）

```
需要坤的時刻：
- 配合他人
- 執行任務
- 支持領導

### 乾坤的平衡

```
理想狀態：

內在有乾（有主見、有能力）
外在用坤（謙虛、配合）

= 謙卦的智慧

反過來：

內在是坤（沒主見、依賴）
外在用乾（強勢、控制）

= 容易出問題
```

## 「用九」與「用六」

乾卦和坤卦各有一個特殊的爻辭：

### 乾卦用九

> 「見群龍無首，吉。」
> 看到群龍都沒有首領，吉。

**解讀：**
- 「無首」不是沒有領導
- 而是每個人都能自我領導
- 不需要一個人站在最前面
- 這是最高境界的領導

### 坤卦用六

> 「利永貞。」
> 利於長久守正。

**解讀：**
- 坤的智慧是「永恆的堅持」
- 不是一時的順從
- 而是長久的德行
