## 第一節：互補指數 — 適度差異法則

### 什麼是互補指數？

每個六十四卦由上卦和下卦組成。我們用「互補指數」衡量它們的差異程度：

```
互補指數 = 上卦與下卦有幾個維度不同

例子：
乾(111) 與 坤(000) = 3個維度不同（完全相反）
乾(111) 與 乾(111) = 0個維度不同（完全相同）
坤(000) 與 艮(001) = 1個維度不同
```

### 驚人發現：2維度差異最佳

我們分析了全部384爻的吉凶數據：

```
差異維度數    樣本數    吉率
───────────────────────────────
0個維度        48      31.2%
1個維度       144      34.7%
2個維度       144      54.2%  ★ 最佳
3個維度        48      41.7%
```

視覺化：

```
吉率
 ↑
60%│            ★ 2維度
   │           ████
50%│           ████
   │           ████
40%│    ████   ████   ████
   │    ████   ████   ████
30%│████████   ████   ████
   └─────────────────────────→
    0維度   1維度   2維度   3維度
    (相同)         (最佳)  (相反)
```

### 為什麼「適度差異」最好？

| 差異程度 | 問題 | 比喻 |
|:--------:|:-----|:-----|
| 0維度 | 缺乏互補，沒有張力 | 兩個一樣的人合作，不會有創新 |
| 3維度 | 衝突太多，難以共識 | 完全相反的人，更可能吵架 |
| 2維度 | 有互補又有共識 | 技能不同但價值觀相同 |

**「求同存異」的量化證明：不是追求完全一致，也不是完全對立，而是「兩個維度不同，一個維度相同」。**

## 第三節：變化穩定度 — Gray碼視角

### 什麼是穩定度？

當一爻變化，吉凶改變多少？這就是「穩定度」。

```
變爻位置    平均吉凶變化    穩定性
──────────────────────────────────
位置1        ±0.15         最穩定
位置2        ±0.20         較穩定
位置3        ±0.45         不穩定 ★
位置4        ±0.25         中等
位置5        ±0.20         較穩定
位置6        ±0.40         不穩定 ★
```

**發現：位置3和位置6變爻最不穩定！**

這印證了「三多凶」和「物極必反」：
- 位置3是轉折點，變化影響大
- 位置6是極端，變化影響大

### 漸進式變化 vs 劇烈變化

```
漸進式（推薦）：
狀態A → 狀態A' → 狀態A'' → 狀態B
每步變化小，可以及時調整，風險可控

劇烈式（高風險）：
狀態A ─────────────→ 狀態B
變化大，難以預測，風險高
```

**易經的智慧：盡量選擇漸進式變化。**

### 卦的變化關係

| 變化類型 | 說明 | 用途 |
|:--------:|:-----|:-----|
| 錯卦 | 陰陽全部互換 | 理解相反狀態 |
| 綜卦 | 上下顛倒 | 理解視角轉換 |
| 互卦 | 取中間四爻 | 理解核心結構 |
| 變卦 | 一爻改變 | 規劃漸進路徑 |

## 實際應用

### 選擇合作夥伴

| 差異程度 | 吉率 | 建議 |
|:--------:|:----:|:-----|
| 找和自己一樣的人 | 31% | 缺乏互補 |
| 找完全相反的人 | 42% | 衝突太多 |
| 找有互補但有共識的人 | 54% | 最佳組合 ★ |

### 評估處境

問自己：
1. 我的外在環境像什麼？（上卦）
2. 我的內在狀態像什麼？（下卦）
3. 它們有幾個維度不同？（互補指數）

```
互補指數    建議
──────────────────────────────
0個維度    尋找互補，增加多元性
1個維度    可以增加一些變化
2個維度    維持現狀，這是好組合 ★
3個維度    找到共識點，求同存異
```

### 規劃變化

在不穩定的位置（3和6）：謹慎決策，準備好應對變化。
在穩定的位置（1、2、5）：可以更有信心地行動。

優先選擇漸進式變化，減少劇烈轉變的風險。
