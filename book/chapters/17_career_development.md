# 第十七章：職業發展——六個階段
# Chapter 17: Career Development — Six Stages

---

## 職業生涯的六爻

一個典型的職業生涯，可以用六爻的階段來理解：

```
位置    職業階段        典型時間        關鍵任務
────────────────────────────────────────────────────
1       新人期          0-2年          學習、適應、觀察
2       成長期          2-5年          建立專業、展現能力
3       轉折期          5-8年          選擇方向、晉升或轉型
4       進階期          8-12年         進入管理或專家軌道
5       成熟期          12-20年        發揮影響、承擔責任
6       傳承期          20年+          培養後輩、考慮退休
```

每個階段有不同的策略。

---

## 位置1：新人期——潛龍勿用

**典型狀態：**
- 剛入職或轉行
- 對環境不熟悉
- 能力還在積累

**正確策略：**

```
✓ 應該做的：
- 多觀察，少評論
- 專注學習，打好基礎
- 建立好關係，了解「潛規則」
- 接受小任務，證明可靠性

✗ 不應該做的：
- 急於表現
- 批評現有做法
- 要求重要項目
- 頻繁發表意見
```

**常見錯誤：**

小李剛入職就在會議上批評公司的技術架構，雖然說得有道理，但得罪了老員工，之後的工作處處受阻。

**教訓：** 在位置1，「對」不代表「該說」。

---

## 位置2：成長期——嶄露頭角

**典型狀態：**
- 熟悉了工作環境
- 開始獨立負責項目
- 有一定的專業能力

**正確策略：**

```
✓ 應該做的：
- 把手上的事做好
- 主動承擔一些額外責任
- 建立可靠的專業形象
- 找一個導師

✗ 不應該做的：
- 過度謙虛，不敢表現
- 過度張揚，搶風頭
- 跳槽太頻繁
- 忽視人際關係
```

**黃金期特徵：**

位置2是職業發展的「黃金期」：
- 已經有基礎了
- 但還沒有到轉折點
- 可以穩定積累
- 吉率46.7%！

---

## 位置3：轉折期——三多凶

**典型狀態：**
- 工作5-8年
- 面臨晉升或瓶頸
- 要做重要的職業選擇

**這是職業生涯最危險的階段！**

```
典型問題：
- 要不要轉管理？
- 要不要換行業？
- 要不要創業？
- 現在的工作還有發展嗎？
```

**正確策略：**

```
✓ 應該做的：
- 深思熟慮，不要衝動
- 多方諮詢，收集信息
- 評估風險，準備備選方案
- 如果要變，要有計劃

✗ 不應該做的：
- 因為焦慮而亂動
- 盲目跟風（別人創業我也創業）
- 同時追求多個方向
- 忽視轉型的代價
```

**數據警告：** 位置3吉率僅6.7%，這個階段做的決定影響深遠。

### 轉折期的兩種路徑

```
路徑A：晉升管理層
- 從做事到管人
- 需要新的技能
- 進入位置4（適應新角色）

路徑B：深耕專業
- 成為專家
- 保持技術能力
- 可能薪資受限但壓力小

路徑C：轉行/創業
- 最高風險
- 需要從位置1重新開始
- 但可能有更大回報
```

---

## 位置4：進階期——近君之位

**典型狀態：**
- 剛升到管理層，或成為資深專家
- 開始接觸更高層
- 需要學習新規則

**正確策略：**

```
✓ 應該做的：
- 謙虛學習管理技能
- 建立跨部門關係
- 理解公司政治
- 從「做事」轉為「成事」

✗ 不應該做的：
- 繼續用之前的方式工作
- 忽視向上管理
- 不授權，事必躬親
- 與同級競爭者搞僵關係
```

**身份轉變：**

從「個人貢獻者」到「管理者」是巨大的轉變：

```
之前：自己做得好就好
現在：要讓團隊做得好

之前：專業能力最重要
現在：領導力、溝通力更重要

之前：專注任務
現在：要思考戰略
```

---

## 位置5：成熟期——九五之尊

**典型狀態：**
- 成為部門負責人或公認專家
- 有話語權和影響力
- 職業進入「舒適區」

**正確策略：**

```
✓ 應該做的：
- 承擔更大責任
- 培養接班人
- 思考更長遠的問題
- 回饋行業/社會

✗ 不應該做的：
- 驕傲自滿
- 停止學習
- 忽視年輕人的崛起
- 只顧保住現有位置
```

**位置5的優勢：**

數據顯示位置5凶率0%——這是職業生涯最安全的階段。

但「安全」不等於「永遠」。要為位置6做準備。

---

## 位置6：傳承期——亢龍有悔

**典型狀態：**
- 職業達到頂峰
- 沒有太多上升空間
- 需要考慮「之後怎麼辦」

**正確策略：**

```
✓ 應該做的：
- 培養接班人
- 分享知識和經驗
- 考慮「第二職業」
- 有尊嚴地準備退出

✗ 不應該做的：
- 戀棧不退
- 打壓年輕人
- 「公司沒有我不行」心態
- 忽視身體健康
```

**智慧的退出：**

位置6最大的智慧是知道何時離開：
- 在還受歡迎的時候離開
- 在還有選擇的時候離開
- 在還能傳承的時候離開

## 換工作的「卦象分析」

用易經思維分析換工作：

### 情境：該不該接受獵頭的邀約？

**分析框架：**

```
1. 你目前的位置
   - 位置1-2：不建議換（還在積累）
   - 位置3：要謹慎評估（轉折期）
   - 位置4-5：看具體情況
   - 位置6：可能是好選擇

2. 新工作讓你進入什麼位置？
   - 如果讓你「降級」回位置1，慎重
   - 如果是平移或升級，可以考慮

3. 上下卦分析
   - 你的內在狀態（下卦）準備好了嗎？
   - 新環境（上卦）適合你嗎？
   - 內外組合是否和諧？
```

### 案例：小陳的選擇

**情況：**
- 小陳在現公司工作3年，是高級工程師
- 獵頭邀請他去創業公司做技術總監
- 薪資漲50%，但要帶團隊

**分析：**

```
現狀：
- 位置2-3（成長到轉折）
- 下卦：艮（想穩定發展）
- 上卦：穩定的大公司

新工作：
- 會進入位置4（新角色適應）
- 下卦：艮（還是他自己）
- 上卦：震（創業公司，變化快）

組合：震上艮下 = 頤卦
頤卦的意思：養育、提供
意味著：他要「養」一個團隊
```

**建議：**
- 如果他內在確實想管人（下卦是震或乾），可以去
- 如果他內在想穩定（下卦是艮），要三思
- 關鍵問題：他有沒有管理的準備？

## 職業發展的「得中」原則

數據顯示，位置2和位置5最好——因為「得中」。

應用到職業發展：

```
「得中」的工作狀態：

✓ 有挑戰但不過度
✓ 有壓力但不崩潰
✓ 有成長但不失控
✓ 有責任但不孤立

「不得中」的警告信號：

⚠ 太舒服——可能在位置1卡住了
⚠ 太痛苦——可能在位置3或6
⚠ 太孤立——可能位置不對
⚠ 太焦慮——可能需要調整
```
