# 第十六章：商業應用：創業與管理

---

## 創業的六個階段

創業是一段旅程，正好對應六爻的六個階段：

```
位置    創業階段        關鍵挑戰
──────────────────────────────────────────────
1       想法期          驗證想法、不要急於辭職
2       啟動期          獲得第一批客戶、建立產品
3       擴張期          管理增長、避免失控 ★危險
4       成熟期          建立系統、專業化管理
5       領導期          市場地位、承擔責任
6       轉型期          尋找第二曲線、考慮退出
```

---

## 位置1：想法期：潛龍勿用

**狀態描述：**
- 有一個商業想法
- 還沒辭職全職創業
- 在驗證可行性

**易經建議：「潛龍勿用」**

```
✓ 正確做法：
- 保留現有工作
- 用業餘時間驗證
- 小規模測試市場
- 找10個目標客戶聊聊

✗ 錯誤做法：
- 立刻辭職
- 租辦公室、招人
- 大規模投入
- 宣傳還沒做好的產品
```

**數據提醒：** 位置1吉率僅26.7%，過早行動風險大。

---

## 位置2：啟動期：得中之位

**狀態描述：**
- 產品已經上線
- 有了第一批客戶
- 開始有收入（或用戶）

**易經建議：穩紮穩打**

```
✓ 正確做法：
- 專注產品質量
- 建立口碑
- 控制成本
- 深入了解客戶需求

✗ 錯誤做法：
- 急於擴張
- 忽視現有客戶
- 過度追求增長數據
```

**數據提醒：** 位置2吉率46.7%，是建立基礎的黃金期。

---

## 位置3：擴張期：三多凶

**狀態描述：**
- 獲得融資或現金流轉正
- 需要快速增長
- 團隊從小變大

**這是最危險的階段！**

```
典型問題：
- 管理跟不上增長
- 燒錢速度過快
- 產品質量下降
- 文化稀釋
- 核心員工離職
```

**易經建議：「夕惕若厲」：保持高度警惕**

```
✓ 正確做法：
- 控制增長速度
- 先建立管理系統
- 保持財務健康
- 保護公司文化

✗ 錯誤做法：
- 盲目追求增長
- 忽視管理問題
- 「燒錢換市場」心態
- 認為「大就是好」
```

**數據提醒：** 位置3吉率僅6.7%！大多數創業公司死在這個階段。

---

## 位置4：成熟期：近君之位

**狀態描述：**
- 公司已有一定規模
- 開始接觸大客戶、投資人
- 需要更專業的管理

**易經建議：謙虛學習新規則**

```
✓ 正確做法：
- 引入專業管理人才
- 建立正規流程
- 學習大企業的運作方式
- 但保持創業精神

✗ 錯誤做法：
- 創始人不願放權
- 抗拒「官僚化」
- 用創業期的方法管理成熟期的公司
```

**數據提醒：** 位置4吉率13.3%，需要適應新的遊戲規則。

---

## 位置5：領導期：九五之尊

**狀態描述：**
- 成為行業領導者或重要玩家
- 有市場地位、品牌影響力
- 公司運轉順暢

**易經建議：承擔責任，不要驕傲**

```
✓ 正確做法：
- 思考社會責任
- 培養接班人
- 為行業做貢獻
- 保持創新，避免僵化

✗ 錯誤做法：
- 驕傲自滿
- 忽視新的競爭者
- 停止創新
- 「我們已經成功了」心態
```

**數據提醒：** 位置5凶率0%，但要記住位置6就在前面。

---

## 位置6：轉型期：亢龍有悔

**狀態描述：**
- 增長見頂
- 市場飽和
- 新的顛覆者出現

**易經建議：知道何時轉型或退出**

```
選項1：尋找第二曲線
- 進入新市場
- 開發新產品
- 併購創新公司

選項2：有尊嚴地退出
- 出售公司
- 傳承給下一代管理者
- 轉型為投資人

✗ 錯誤做法：
- 死守原有業務
- 與趨勢對抗
- 「公司就是我的命」
```

**數據提醒：** 位置6凶率40%，物極必反。

---

## 管理中的「位置意識」

作為管理者，你需要理解團隊成員的「位置」：

### 不同位置的下屬

```
位置1的下屬（新人）
- 需要：指導、培訓、耐心
- 不要：給太多責任、急於要結果

位置2的下屬（成長中）
- 需要：機會、支持、反饋
- 不要：過度干預、忽視他們

位置3的下屬（轉折點）
- 需要：關注、輔導、幫助決策
- 不要：忽視他們的掙扎

位置5的下屬（核心骨幹）
- 需要：授權、尊重、資源
- 不要：微管理、搶功勞
```

### 你自己的位置

作為管理者，問自己：
- 我在這個組織中處於什麼位置？
- 這個位置適合什麼行為？
- 我有沒有「逆位」（做不適合這個位置的事）？

## 商業決策的「卦象分析」

用上下卦分析商業決策：

### 案例：是否接受大客戶的獨家協議

**情境：**
- 一個大客戶願意簽大單，但要求獨家供應
- 這會讓你獲得穩定收入，但失去其他客戶機會

**卦象分析：**

```
如果接受：
- 下卦（你的狀態）= 坤（順從）
- 上卦（外在環境）= 乾（強勢客戶主導）
- 組合 = 天地否？還是地天泰？

關鍵問題：
- 你是「順從」還是「配合」？
- 這種關係是「不交」還是「交泰」？
```

**決策建議：**
- 如果你處於位置1-2（還在成長），接受可能有益（獲得穩定）
- 如果你處於位置3-5（有一定實力），要謹慎（可能限制發展）
- 談判條件很重要：能否保留部分獨立性？

## 創業者的「八卦性格」

用八卦理解創業團隊的角色：

```
══════════════════════════════════════════════════
                  創業團隊的八卦角色                  
├──────┬───────────────────────────────────────┤
   乾      CEO/創始人 — 願景、決策、領導             
   坤      COO/運營 — 執行、協調、落地              
   艮      CFO/風控 — 底線、節制、風險管理           
   兌      銷售/公關 — 溝通、談判、客戶關係          
   離      品牌/設計 — 形象、視覺、被看見            
   坎      策略/分析 — 洞察、智慧、危機管理          
   震      產品/創新 — 突破、行動、新功能            
   巽      市場/增長 — 滲透、漸進、用戶增長          
└──────┴───────────────────────────────────────┘
```

**理想的團隊組合：**
- 需要有「乾」（領導）和「坤」（執行）
- 需要有「艮」（底線）防止失控
- 需要有「震」（創新）和「巽」（滲透）來增長
