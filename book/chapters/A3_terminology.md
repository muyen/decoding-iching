## 基本概念

| 中文術語 | 英文 | 說明 |
|---------|------|------|
| 陰 | Yin | 斷開的線（━ ━），代表0、柔、靜 |
| 陽 | Yang | 完整的線（━━━），代表1、剛、動 |
| 爻 | Yao/Line | 卦的基本單位，一條線 |
| 卦 | Gua/Hexagram | 由六爻組成的符號 |
| 八卦 | Bagua/Trigrams | 三爻組成的8種基本卦 |
| 六十四卦 | 64 Hexagrams | 六爻組成的64種卦 |

## 數字對照

### 八卦數值

```
坤=0, 艮=1, 坎=2, 巽=3
震=4, 離=5, 兌=6, 乾=7
```

### 位置數值

```
初爻=1, 二爻=2, 三爻=3
四爻=4, 五爻=5, 上爻=6
```

### 二進制

```
陰(━ ━)=0
陽(━━━)=1

例如：乾=111(7), 坤=000(0), 艮=001(1)
```

## 常見問答

**Q: 「九」和「六」是什麼意思？**
A: 「九」代表陽爻，「六」代表陰爻。如「九五」指位置5的陽爻。

**Q: 什麼是「得中」？**
A: 處於下卦中間（位置2）或上卦中間（位置5），代表不偏不倚。

**Q: 什麼是「當位」？**
A: 陽爻在奇數位（1、3、5），陰爻在偶數位（2、4、6），叫當位。

**Q: 「用九」「用六」是什麼？**
A: 乾卦和坤卦獨有的附加爻辭，代表所有爻都動時的情況。
