## 發現2：XOR規則（XOR Pattern Discovery）

### 這是什麼？

XOR（異或）是計算機科學中的基本運算。我們發現上卦和下卦的XOR值能預測吉凶。

```
XOR計算：
上卦 ⊕ 下卦 = 差異值（0-7）

例如：
乾(111) ⊕ 坤(000) = 111 = 7
乾(111) ⊕ 乾(111) = 000 = 0（純卦）
```

### XOR吉凶分布

```
XOR值   樣本數   吉率    平均分    特殊發現
────────────────────────────────────────────
  0       12    33.3%   +0.17    純卦，中位必吉
  1       12    16.7%   +0.00
  2       12    16.7%   -0.25
  3       12    16.7%   +0.00
  4       12    50.0%   +0.33    前4爻100%吉！
  5       12    25.0%   +0.08
  6        6    16.7%   +0.00
  7       12    25.0%   +0.08
```

### 突破性發現：適度差異法則

```
┌──────────────────────────────────────────────────┐
│                                                  │
│   2維度差異 → 54% 吉率（最佳）                   │
│   0維度差異 → 31% 吉率（太相似）                 │
│   3維度差異 → 42% 吉率（太對立）                 │
│                                                  │
│   這是「求同存異」的量化證明！                   │
│                                                  │
└──────────────────────────────────────────────────┘
```

**解讀：** 適度差異（2個維度不同，1個維度相同）比完全一致或完全相反更吉利。

## 發現6：Gray碼穩定度

### 方法

Gray碼是一種二進制編碼，相鄰數字只差一位。我們用它來測量卦的「穩定度」。

```
Gray碼轉換：
二進制 → Gray碼
110000 → 101000 (Gray碼有2個1)
111111 → 100000 (Gray碼有1個1)
010101 → 011111 (Gray碼有5個1)
```

### 統計結果

```
Gray碼1的數量   吉率    含義
────────────────────────────────
    ≤2         38.9%   結構穩定 ✓
    3-4        20.8%   變動中
    ≥5         16.7%   不穩定 ✗
```

### 關鍵洞察

> **Gray碼1越少，結構越穩定，吉率越高。** 這提供了一個量化「卦象穩定度」的方法。

## 發現9：全新發現

### 外卦優於內卦

```
位置類別    吉率    凶率    淨率
────────────────────────────────────
內卦(1-3爻)  31.8%   19.8%   +12.0%
外卦(4-6爻)  37.5%   13.5%   +24.0%
```

**結論：** 外卦（4-6爻）整體優於內卦（1-3爻）。

### 錯卦關係

```
傳統關係    相關係數    說明
────────────────────────────────────
綜卦（顛倒）   -0.021   無明顯關聯
錯卦（陰陽換）  +0.457   正相關
互卦（內在）   +0.092   弱正相關
```

**發現：** 錯卦整體呈正相關（r=+0.46）。但需注意：
- 經典對立卦（坎↔離、震↔巽）確實吉凶相反，符合傳統
- 其他多數錯卦對吉凶傾向相似
- 「性質對立」不等於「吉凶對立」

### 動爻標籤對變卦的影響

```
動爻標籤    平均變化
────────────────────
吉          -0.89
中          +0.26
凶          +1.11
```

**發現：** 動爻為凶時，變卦後更可能改善！困境中變化是好事。

### 異卦優於同卦

```
組合類型    平均淨分
────────────────────
同卦組合     +0.12
異卦組合     +1.21
```

**發現：** 上下卦不同比相同更好，差異產生價值。

### 六五優於九五

**發現：** 陰爻在五位（六五）比陽爻在五位（九五）更吉！柔中帶剛。
