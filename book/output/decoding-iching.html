前言：易經最大的秘密

一個顛覆常識的發現

三千年來，人們追求「吉」、避開「凶」。

但什麼是「吉」？

我們用統計方法分析了易經384爻，發現了一個顛覆常識的答案：

乾卦：最強大的卦：吉率是 0%。

六爻全陽，代表極致的力量、創造力、領導力。按常理，這應該是最好的狀態。

但數據顯示：乾卦沒有一爻是吉的。

為什麼？

吉不是「好」，是「還能變」

答案就藏在「易」這個字裡。

易 = 變

易經的核心不是告訴你什麼狀態「好」，而是告訴你：

能夠繼續變化的狀態，就是好狀態。

乾卦為什麼 0% 吉率？因為它滿了。六爻全陽，沒有上升空間，沒有發展餘地。

謙卦為什麼 83% 吉率？因為它空著。高山藏在地下，有無限的上升空間。

這就是易經的秘密：

═══════════════════════════════════════════

  吉 ≠ 最好的狀態
  吉 = 最有發展可能的狀態

  滿了 = 凶
  有餘地 = 吉

═══════════════════════════════════════════

這顛覆了什麼？

社會告訴我們：

  • 爬到頂峰
  • 成為最強
  • 累積更多
  • 贏得勝利

易經告訴我們：

  • 頂峰 = 沒有上升空間 = 危險
  • 最強 = 無法再強 = 轉折點
  • 太滿 = 沒有餘地 = 凶
  • 贏到底 = 窮途末路

目標不是贏。目標是留在遊戲裡。

古人早就知道

這不是我們發明的。這是我們揭開的。

2500年前，繫辭傳就寫了：

「窮則變，變則通，通則久。」

  • 窮 = 被卡住 = 凶（我們的數據：第三爻 10.9% 吉率）
  • 變 = 能變 = 有選擇
  • 通 = 流動 = 吉（我們的數據：第五爻 48.4% 吉率）
  • 久 = 持續發展

64卦為什麼以「未濟」結束，而不是「既濟」？

因為「完成」= 沒有變化空間 = 凶。 「未完成」= 還有可能 = 吉的前提。

易經從來不是教你追求完美。易經是教你保持可能性。

最驚人的發現：視角決定命運

我們發現了一對卦，讓這個原理變得無比清晰。

臨卦和觀卦是「綜卦」：同一個卦上下顛倒。

卦 吉率 意義

陰陽是什麼？

陽 用一條完整的橫線表示：⚊（—） 陰 用一條中斷的橫線表示：⚋（- -）

這不是說陽好陰壞。兩者都是必要的，相互依存。

沒有黑夜，就沒有白天。 沒有靜止，就沒有行動。 沒有失敗，就沒有成功。

為什麼用二進制？

簡單

兩個選項意味著最少的判斷：是或否，有或無，開或關。

複雜的事物可以分解成多個二元選擇： - 這個機會大於風險嗎？（是/否） - 現在是行動的時機嗎？（是/否） - 我準備好了嗎？（是/否）

第四章：六十四卦：組合的力量

從字母到單詞

上一章我們認識了8個「字母」（八卦）。

現在，把兩個字母疊在一起，會產生多少種組合？

8 × 8 = 64 種

這就是六十四卦：易經的64個「單詞」。

卦的結構

每個六十四卦由兩部分組成：

    ━━━  ─┐
    ━ ━   ├─ 上卦（外卦）
    ━━━  ─┘
    ━━━  ─┐
    ━ ━   ├─ 下卦（內卦）
    ━━━  ─┘

上卦：代表外在、環境、他人 下卦：代表內在、自己、基礎

用數學表示：

卦 = (上卦, 下卦) = (0-7, 0-7)

例如： - 乾卦 = (7, 7) = 乾上乾下 = 「77」 - 坤卦 = (0, 0) = 坤上坤下 = 「00」 - 謙卦 = (0, 1) = 坤上艮下 = 「01」

單詞的意義

就像英文單詞 “breakfast” 不只是 “break” + “fast”，六十四卦的意義也超越了上下卦的簡單相加。

例子：謙卦

上卦：坤（地）= 順從、低下
下卦：艮（山）= 高大、停止

組合：山在地下
意義：高山藏於地下，不顯露：這就是「謙虛」

高大的山本應高聳，卻甘於在地下。這種姿態就是謙。

例子：否卦

上卦：乾（天）
下卦：坤（地）

組合：天在上，地在下
看起來正常？但...
意義：天地不交，閉塞不通：這就是「否」

為什麼？因為天氣上升、地氣下降，各走各的，無法交流。

相反，泰卦是坤上乾下（地在上、天在下），代表天地交泰：天氣下降、地氣上升，相交而通。

這就是易經的辯證思維：表面的「正確」不一定好，看似「顛倒」反而通泰。

西洋棋與易經

想像一個8×8的棋盤。

這不是用來下棋的，而是用來看清你的處境。

每個格子是一個卦，由兩個坐標決定： - X軸（橫軸）= 上卦 = 外在環境 - Y軸（縱軸）= 下卦 = 內在狀態

這個視角讓我們能用空間思維理解64卦的關係。

為什麼用棋盤？

傳統上，64卦被排成各種方陣（如先天方圖、後天方圖）。但那些排列有其神秘學背景，不容易用現代思維理解。

我們的棋盤排列有一個清晰的邏輯：

坐標(x, y) = (上卦, 下卦) = (外在, 內在)

這樣排列的好處：

  1. 直觀理解：橫向移動 = 改變外在環境；縱向移動 = 改變內在狀態
  2. 發現規律：相鄰的卦有相似的結構，可以看出漸變關係
  3. 策略思考：可以像下棋一樣思考「往哪個方向走」

四象限分析

把棋盤分成四個象限：

                  左半 (x<4)          右半 (x≥4)
                ─────────────────  ─────────────────

  上半 (y≥4)        Q2                  Q1

                  較保守              最佳區域
                 內強外弱            內外皆強
                ─────────────────  ─────────────────

  下半 (y<4)        Q3                  Q4

                  最弱區域            較激進
                 內外皆弱            外強內弱
                ─────────────────  ─────────────────

我們的數據分析發現：

象限    位置      吉率      特徵
────────────────────────────────────────
Q1      右上      38.9%     內外皆強，最好
Q2      左上      33.3%     內強外弱，穩健
Q3      左下      20.0%     內外皆弱，最差
Q4      右下      20.8%     外強內弱，不穩

關鍵發現：上半區（下卦值≥4）整體較吉。

這意味著什麼？下卦代表內在基礎。內在強大是好結果的前提。

純卦代表內外一致，但有趣的是：

這提示我們：完全一致不一定最好。數據顯示，有適度差異（2個維度不同）的卦反而吉率更高（54%）。詳見第十章。

移動規則：往哪走更好？

如果你目前的處境在棋盤某個位置，往哪個方向「移動」會更好？

我們分析了所有相鄰卦之間的吉凶變化：

移動方向          平均吉凶變化      含義
───────────────────────────────────────────────
向右移動 (x+1)      +1.00         傾向變好 ✓
向左移動 (x-1)      -1.00         傾向變差 ✗
向上移動 (y+1)       0.00         無明顯影響
向下移動 (y-1)       0.00         無明顯影響

驚人發現：橫向移動（改變上卦）比縱向移動（改變下卦）影響更大！

這意味著什麼？

  • 上卦 = 外在環境
  • 改變外在環境比改變內在狀態更能直接影響結果

這聽起來可能違反直覺：我們不是常說「改變自己比改變環境容易」嗎？

但易經的洞見是：環境選擇的重要性被低估了。

實際應用：環境選擇

讓我們用棋盤思維分析一個職業決定：

情境：選擇工作環境

小王有兩個工作機會： - A公司：大企業，穩定但僵化（坤的特質） - B公司：創業公司，有活力但風險高（乾的特質）

小王自己的狀態：剛畢業，有能力但經驗少（艮的特質：需要停下來學習）

用棋盤分析：

選擇A：小王(艮) + A公司(坤) = 坤上艮下 = 謙卦
選擇B：小王(艮) + B公司(乾) = 乾上艮下 = 大畜卦

謙卦（坤上艮下）是吉率很高的卦（67%）。 大畜卦（乾上艮下）也是好卦，但需要積累才能行動。

對於經驗不足的小王，選擇穩定的環境先積累，可能比跳入高風險環境更明智。

這就是棋盤思維的威力：把抽象的選擇變成可視化的位置分析。

XOR視角：差異度分析

棋盤上還有一個隱藏的維度：上下卦的差異度。

用XOR（異或運算）計算：

XOR = 上卦 ⊕ 下卦

例如：
乾(111) ⊕ 坤(000) = 111 = 7（最大差異）
乾(111) ⊕ 乾(111) = 000 = 0（零差異，純卦）
坤(000) ⊕ 艮(001) = 001 = 1（最小差異）

在棋盤上，XOR值形成一個有趣的圖案：

XOR值 = 0 → 對角線（純卦）
XOR值 = 7 → 反對角線（最大對比）

我們發現：

差異維度    XOR值       吉率
─────────────────────────────
0維度      XOR=0       31%   太相似
1維度      XOR=1,2,4   35%   差異不足
2維度      XOR=3,5,6   54%   最佳 ★
3維度      XOR=7       42%   太對立

核心發現：2維度差異最佳。 這是「求同存異」的量化證明：太相似不好，太對立也不好，適度差異最好。詳見第十章。

如何使用棋盤

Step 1：確定你的位置

問自己： - 我的內在狀態像哪個八卦？（下卦） - 外在環境像哪個八卦？（上卦）

Step 2：在棋盤上找到你的卦

根據(上卦, 下卦)坐標找到位置。

Step 3：評估處境

  • 你在哪個象限？
  • 離吉利區域多遠？
  • 周圍的「鄰居」是什麼卦？

Step 4：規劃移動

  • 向右移動（改變環境）傾向變好
  • 向上移動（提升內在）打好基礎
  • 避免左下角區域

「三多凶」的心理分析

在第三爻，人們通常會犯的錯誤：

  1. 驕傲自滿：在下卦已經「成功」，開始輕敵
  2. 急於求成：想快速跨入上卦，操之過急
  3. 進退失據：留在原地沒發展，向前又沒把握

乾卦第三爻說：「君子終日乾乾，夕惕若厲，無咎」

這正是第三爻的生存指南：加倍謹慎。

漸進式變化 vs 劇烈變化

漸進式（推薦）：
狀態A → 狀態A' → 狀態A'' → 狀態B
每步變化小，可以及時調整，風險可控

劇烈式（高風險）：
狀態A ─────────────→ 狀態B
變化大，難以預測，風險高

易經的智慧：盡量選擇漸進式變化。

  • 「吝」 從來不是凶，只是「有點可惜」
  • 「利」 不如想像中可靠

條件句的結構

有些爻辭同時有「吉」和「凶」：

「征凶，居貞吉」
  → 打仗會凶，待在家裡會吉

這是條件句，不是「又吉又凶」。

古人的寫作習慣似乎是「重點放後面」：最後出現的詞往往是結論。

從理論到實踐

前面13章，我們學習了易經的結構和規律。

核心原則：吉 = 選擇權

在進入五步法之前，記住這個最重要的原則：

吉 = 變化的空間 = 自由 = 選擇權

這就像： - 金錢 — 可以換成任何你需要的東西 - 自由 — 可以走任何你想走的路 - 健康 — 可以做任何你想做的事

易經的「吉」，就是這種「選擇的可能性」。

沒有選擇 = 窮 = 凶 有選擇 = 通 = 吉

所以在做決策時，核心問題是：

這個選擇會增加還是減少我的選擇空間？

選擇空間增加 → 傾向做 選擇空間減少 → 謹慎考慮

第十五章：時機的藝術：何時動何時靜

易經的核心：時機

如果用一個詞概括易經的智慧，那就是：時機。

不是「做什麼」，而是「什麼時候做」。

同樣的行動，在不同的時機，結果可能完全相反。

六個時機階段

我們已經知道六爻代表六個階段：

位置    時機        關鍵詞      最佳行動
──────────────────────────────────────────────
1       潛伏期      勿用        等待、積累
2       成長期      展現        穩健發展
3       轉折期      謹慎        小心決策
4       適應期      學習        觀察適應
5       巔峰期      發揮        承擔責任
6       極端期      知退        轉型離開

每個時機都有它的「正確打開方式」。

「動」與「靜」的時機

什麼時候該「動」？

適合行動的條件：

✓ 位置2或5（得中）
✓ 外在環境支持（上卦是乾或坤）
✓ 內在準備好了（下卦穩固）
✓ XOR值適中（不是極端）
✓ 已經積累了足夠的基礎

什麼時候該「靜」？

適合等待的條件：

✓ 位置1（剛開始）
✓ 位置3（轉折點）
✓ 位置6（到了極端）
✓ 內外不和諧（XOR高）
✓ 外在環境不利（上卦是坎或兌）

案例分析：投資時機

情境1：牛市初期

市場狀態：剛從低點回升
你的狀態：觀望中，有些資金

位置分析：
- 市場在位置1-2（剛開始成長）
- 你的心態是「艮」（想等等看）

時機判斷：
- 位置1-2是好的進場時機
- 但不要All in，逐步建倉

情境2：市場瘋狂

市場狀態：連續大漲，人人談股
你的狀態：害怕錯過，想追高

位置分析：
- 市場在位置5-6（接近頂部）
- 你的心態是「兌」（追求愉悅的回報）

時機判斷：
- 位置6「亢龍有悔」
- 這時候應該「靜」，而不是「動」
- 獲利了結比追高更明智

情境3：恐慌時刻

市場狀態：大跌，人心惶惶
你的狀態：恐懼，想賣出

位置分析：
- 市場在位置3（轉折點）
- 你的心態是「坎」（感到險阻）

時機判斷：
- 位置3確實危險
- 但「物極必反」：最恐慌可能是低點
- 謹慎評估，不要恐慌性賣出

「時義」：每個卦的時機含義

每個六十四卦都有自己的「時義」：這個卦代表什麼樣的時機。

幾個重要的時義

乾卦 — 創造的時機
  適合：開創、領導、積極進取
  注意：不要過度（亢龍有悔）

坤卦 — 順從的時機
  適合：配合、等待、做好支持
  注意：不要有自己的主張

屯卦 — 艱難開始的時機
  適合：打基礎、不要急
  注意：開始總是艱難的，堅持住

蒙卦 — 學習的時機
  適合：請教、學習、接受指導
  注意：不恥下問

需卦 — 等待的時機
  適合：耐心等待時機成熟
  注意：等待不是無所事事

訟卦 — 爭執的時機
  適合：尋求仲裁，不要硬碰硬
  注意：爭訟無益

謙卦 — 謙虛的時機
  適合：低調、謙虛、不出頭
  注意：吉率最高的卦之一（83.3%）

豫卦 — 愉悅的時機
  適合：享受、娛樂、放鬆
  注意：不要過度享樂

識別時機轉換的信號

時機不是靜態的，它會轉換。識別轉換信號很重要。

從「潛伏」到「成長」的信號

✓ 開始有人關注你
✓ 小的成果出現
✓ 獲得第一個機會
✓ 感覺「準備好了」

從「成長」到「轉折」的信號

✓ 增長開始放緩
✓ 遇到瓶頸
✓ 需要做重大決定
✓ 舊方法不管用了

從「巔峰」到「極端」的信號

✓ 沒有新的挑戰
✓ 開始感到無聊
✓ 後來者在追趕
✓ 聽到「你已經是最好的」

時機的「節奏感」

易經教的不是「公式」，而是「節奏感」。

快與慢的節奏

該快的時候（陽性時機）：
- 機會出現，果斷行動
- 競爭激烈，快速響應
- 處於上升期，乘勝追擊

該慢的時候（陰性時機）：
- 形勢不明，觀察等待
- 剛到新環境，謹慎適應
- 處於轉折點，深思熟慮

進與退的節奏

該進的時候：
- 位置2、4、5（有發展空間）
- 外在環境支持
- 內在準備充分

該退的時候：
- 位置3、6（轉折或極端）
- 外在環境不利
- 內在疲憊或不確定

實際應用：一天的時機

把時機概念應用到一天的工作中：

早晨（位置1-2）
- 潛伏→成長
- 適合：規劃、準備、啟動重要工作

上午（位置2-3）
- 成長→轉折
- 適合：處理核心任務、做決策

中午（位置3-4）
- 轉折→適應
- 適合：休息、調整、反思上午的工作

下午（位置4-5）
- 適應→巔峰
- 適合：協作、會議、發揮影響力

傍晚（位置5-6）
- 巔峰→極端
- 適合：收尾、總結、不要開始新項目

晚上（位置6→1）
- 極端→新的潛伏
- 適合：休息、反思、為明天準備

「不占而知」的境界

古人說：「善易者不占」：真正理解易經的人不需要占卦。

為什麼？因為他們已經內化了時機的感覺：

他們能感知：
- 現在是什麼時機
- 下一步可能如何變化
- 什麼是合適的行動

他們的習慣：
- 持續觀察
- 保持覺知
- 隨時調整

這是一種「時機直覺」：經過練習，你也可以發展出來。

每日反思

今天結束時問自己：
1. 今天我處於什麼「位置」？
2. 我的行動是否符合這個時機？
3. 有什麼信號提示時機正在轉換？

每週回顧

這週結束時問自己：
1. 這週整體是什麼時機？（潛伏/成長/轉折等）
2. 我是「順時」還是「逆時」？

重大決策前

面對重大決策時問自己：
1. 現在是做這個決定的好時機嗎？
2. 如果不是，什麼時候可能更好？
3. 如果必須現在做，如何降低時機不對的風險？

   □ 我有能力、有信心 → +1分

   □ 其實沒把握 → -1分

0-1分 → 再準備一下

第十六章：商業應用：創業與管理

創業的六個階段

創業是一段旅程，正好對應六爻的六個階段：

位置    創業階段        關鍵挑戰
──────────────────────────────────────────────
1       想法期          驗證想法、不要急於辭職
2       啟動期          獲得第一批客戶、建立產品
3       擴張期          管理增長、避免失控 ★危險
4       成熟期          建立系統、專業化管理
5       領導期          市場地位、承擔責任
6       轉型期          尋找第二曲線、考慮退出

位置1：想法期：潛龍勿用

狀態描述： - 有一個商業想法 - 還沒辭職全職創業 - 在驗證可行性

易經建議：「潛龍勿用」

✓ 正確做法：
- 保留現有工作
- 用業餘時間驗證
- 小規模測試市場
- 找10個目標客戶聊聊

✗ 錯誤做法：
- 立刻辭職
- 租辦公室、招人
- 大規模投入
- 宣傳還沒做好的產品

數據提醒： 位置1吉率僅26.7%，過早行動風險大。

位置2：啟動期：得中之位

狀態描述： - 產品已經上線 - 有了第一批客戶 - 開始有收入（或用戶）

易經建議：穩紮穩打

✓ 正確做法：
- 專注產品質量
- 建立口碑
- 控制成本
- 深入了解客戶需求

✗ 錯誤做法：
- 急於擴張
- 忽視現有客戶
- 過度追求增長數據

數據提醒： 位置2吉率46.7%，是建立基礎的黃金期。

位置3：擴張期：三多凶

狀態描述： - 獲得融資或現金流轉正 - 需要快速增長 - 團隊從小變大

這是最危險的階段！

典型問題：
- 管理跟不上增長
- 燒錢速度過快
- 產品質量下降
- 文化稀釋
- 核心員工離職

易經建議：「夕惕若厲」：保持高度警惕

✓ 正確做法：
- 控制增長速度
- 先建立管理系統
- 保持財務健康
- 保護公司文化

✗ 錯誤做法：
- 盲目追求增長
- 忽視管理問題
- 「燒錢換市場」心態
- 認為「大就是好」

數據提醒： 位置3吉率僅6.7%！大多數創業公司死在這個階段。

位置4：成熟期：近君之位

狀態描述： - 公司已有一定規模 - 開始接觸大客戶、投資人 - 需要更專業的管理

易經建議：謙虛學習新規則

✓ 正確做法：
- 引入專業管理人才
- 建立正規流程
- 學習大企業的運作方式
- 但保持創業精神

✗ 錯誤做法：
- 創始人不願放權
- 抗拒「官僚化」
- 用創業期的方法管理成熟期的公司

數據提醒： 位置4吉率13.3%，需要適應新的遊戲規則。

位置5：領導期：九五之尊

狀態描述： - 成為行業領導者或重要玩家 - 有市場地位、品牌影響力 - 公司運轉順暢

易經建議：承擔責任，不要驕傲

✓ 正確做法：
- 思考社會責任
- 培養接班人
- 為行業做貢獻
- 保持創新，避免僵化

✗ 錯誤做法：
- 驕傲自滿
- 忽視新的競爭者
- 停止創新
- 「我們已經成功了」心態

數據提醒： 位置5凶率0%，但要記住位置6就在前面。

位置6：轉型期：亢龍有悔

狀態描述： - 增長見頂 - 市場飽和 - 新的顛覆者出現

易經建議：知道何時轉型或退出

選項1：尋找第二曲線
- 進入新市場
- 開發新產品
- 併購創新公司

選項2：有尊嚴地退出
- 出售公司
- 傳承給下一代管理者
- 轉型為投資人

✗ 錯誤做法：
- 死守原有業務
- 與趨勢對抗
- 「公司就是我的命」

數據提醒： 位置6凶率40%，物極必反。

第十七章：職業發展：六個階段

職業生涯的六爻

一個典型的職業生涯，可以用六爻的階段來理解：

位置    職業階段        典型時間        關鍵任務
────────────────────────────────────────────────────
1       新人期          0-2年          學習、適應、觀察
2       成長期          2-5年          建立專業、展現能力
3       轉折期          5-8年          選擇方向、晉升或轉型
4       進階期          8-12年         進入管理或專家軌道
5       成熟期          12-20年        發揮影響、承擔責任
6       傳承期          20年+          培養後輩、考慮退休

每個階段有不同的策略。

位置1：新人期：潛龍勿用

典型狀態： - 剛入職或轉行 - 對環境不熟悉 - 能力還在積累

正確策略：

✓ 應該做的：
- 多觀察，少評論
- 專注學習，打好基礎
- 建立好關係，了解「潛規則」
- 接受小任務，證明可靠性

✗ 不應該做的：
- 急於表現
- 批評現有做法
- 要求重要項目
- 頻繁發表意見

常見錯誤：

小李剛入職就在會議上批評公司的技術架構，雖然說得有道理，但得罪了老員工，之後的工作處處受阻。

教訓： 在位置1，「對」不代表「該說」。

位置2：成長期：嶄露頭角

典型狀態： - 熟悉了工作環境 - 開始獨立負責項目 - 有一定的專業能力

正確策略：

✓ 應該做的：
- 把手上的事做好
- 主動承擔一些額外責任
- 建立可靠的專業形象
- 找一個導師

✗ 不應該做的：
- 過度謙虛，不敢表現
- 過度張揚，搶風頭
- 跳槽太頻繁
- 忽視人際關係

黃金期特徵：

位置2是職業發展的「黃金期」： - 已經有基礎了 - 但還沒有到轉折點 - 可以穩定積累 - 吉率46.7%！

位置3：轉折期：三多凶

典型狀態： - 工作5-8年 - 面臨晉升或瓶頸 - 要做重要的職業選擇

這是職業生涯最危險的階段！

典型問題：
- 要不要轉管理？
- 要不要換行業？
- 要不要創業？
- 現在的工作還有發展嗎？

正確策略：

✓ 應該做的：
- 深思熟慮，不要衝動
- 多方諮詢，收集信息
- 評估風險，準備備選方案
- 如果要變，要有計劃

✗ 不應該做的：
- 因為焦慮而亂動
- 盲目跟風（別人創業我也創業）
- 同時追求多個方向
- 忽視轉型的代價

數據警告： 位置3吉率僅6.7%，這個階段做的決定影響深遠。

轉折期的兩種路徑

路徑A：晉升管理層
- 從做事到管人
- 需要新的技能
- 進入位置4（適應新角色）

路徑B：深耕專業
- 成為專家
- 保持技術能力
- 可能薪資受限但壓力小

路徑C：轉行/創業
- 最高風險
- 需要從位置1重新開始
- 但可能有更大回報

位置4：進階期：近君之位

典型狀態： - 剛升到管理層，或成為資深專家 - 開始接觸更高層 - 需要學習新規則

正確策略：

✓ 應該做的：
- 謙虛學習管理技能
- 建立跨部門關係
- 理解公司政治
- 從「做事」轉為「成事」

✗ 不應該做的：
- 繼續用之前的方式工作
- 忽視向上管理
- 不授權，事必躬親
- 與同級競爭者搞僵關係

身份轉變：

從「個人貢獻者」到「管理者」是巨大的轉變：

之前：自己做得好就好
現在：要讓團隊做得好

之前：專業能力最重要
現在：領導力、溝通力更重要

之前：專注任務
現在：要思考戰略

位置5：成熟期：九五之尊

典型狀態： - 成為部門負責人或公認專家 - 有話語權和影響力 - 職業進入「舒適區」

正確策略：

✓ 應該做的：
- 承擔更大責任
- 培養接班人
- 思考更長遠的問題
- 回饋行業/社會

✗ 不應該做的：
- 驕傲自滿
- 停止學習
- 忽視年輕人的崛起
- 只顧保住現有位置

位置5的優勢：

數據顯示位置5凶率0%：這是職業生涯最安全的階段。

但「安全」不等於「永遠」。要為位置6做準備。

位置6：傳承期：亢龍有悔

典型狀態： - 職業達到頂峰 - 沒有太多上升空間 - 需要考慮「之後怎麼辦」

正確策略：

✓ 應該做的：
- 培養接班人
- 分享知識和經驗
- 考慮「第二職業」
- 有尊嚴地準備退出

✗ 不應該做的：
- 戀棧不退
- 打壓年輕人
- 「公司沒有我不行」心態
- 忽視身體健康

智慧的退出：

位置6最大的智慧是知道何時離開： - 在還受歡迎的時候離開 - 在還有選擇的時候離開 - 在還能傳承的時候離開

換工作的「卦象分析」

用易經思維分析換工作：

情境：該不該接受獵頭的邀約？

分析框架：

1. 你目前的位置
   - 位置1-2：不建議換（還在積累）
   - 位置3：要謹慎評估（轉折期）
   - 位置4-5：看具體情況
   - 位置6：可能是好選擇

2. 新工作讓你進入什麼位置？
   - 如果讓你「降級」回位置1，慎重
   - 如果是平移或升級，可以考慮

3. 上下卦分析
   - 你的內在狀態（下卦）準備好了嗎？
   - 新環境（上卦）適合你嗎？
   - 內外組合是否和諧？

案例：小陳的選擇

情況： - 小陳在現公司工作3年，是高級工程師 - 獵頭邀請他去創業公司做技術總監 - 薪資漲50%，但要帶團隊

分析：

現狀：
- 位置2-3（成長到轉折）
- 下卦：艮（想穩定發展）
- 上卦：穩定的大公司

新工作：
- 會進入位置4（新角色適應）
- 下卦：艮（還是他自己）
- 上卦：震（創業公司，變化快）

組合：震上艮下 = 頤卦
頤卦的意思：養育、提供
意味著：他要「養」一個團隊

建議： - 如果他內在確實想管人（下卦是震或乾），可以去 - 如果他內在想穩定（下卦是艮），要三思 - 關鍵問題：他有沒有管理的準備？

職業發展的「得中」原則

數據顯示，位置2和位置5最好：因為「得中」。

應用到職業發展：

「得中」的工作狀態：

✓ 有挑戰但不過度
✓ 有壓力但不崩潰
✓ 有成長但不失控
✓ 有責任但不孤立

「不得中」的警告信號：

⚠ 太舒服：可能在位置1卡住了
⚠ 太痛苦：可能在位置3或6
⚠ 太孤立：可能位置不對
⚠ 太焦慮：可能需要調整

第十八章：人際關係：位置意識

關係中的「位置」

每段關係中，雙方都有自己的「位置」。

這不是說誰高誰低，而是說： - 每個人在關係中扮演什麼角色 - 這個角色適合什麼行為 - 角色之間如何互動

理解位置，可以避免很多關係中的問題。

上下卦：關係的結構

在關係分析中： - 下卦 = 你自己的狀態 - 上卦 = 對方的狀態，或關係的環境

例如：

你在新戀情中感到不安（坎=險）
對方很強勢、主導（乾=天）
關係結構 = 乾上坎下 = 訟卦

訟卦的意思：爭執、需要仲裁
暗示：這段關係可能有權力不平衡的問題

親密關係的六個階段

用六爻理解戀愛/婚姻的發展：

位置    關係階段        特徵
────────────────────────────────────────────
1       相識期          剛認識，互相觀察
2       熱戀期          感情升溫，建立連結
3       磨合期          現實問題出現，衝突增加
4       穩定期          學會相處，形成默契
5       成熟期          深厚信任，互相支持
6       轉變期          關係需要更新或結束

位置1：相識期

狀態： 剛認識，還在了解對方

正確做法： - 保持好奇心 - 不要太快下結論 - 展現真實的自己 - 觀察對方的行為而非言語

錯誤做法： - 急於確定關係 - 過度投入 - 忽視警告信號

位置2：熱戀期

狀態： 感情迅速升溫

正確做法： - 享受這個階段 - 建立共同經歷 - 開始了解深層價值觀 - 保持個人獨立性

錯誤做法： - 忽略對方的缺點 - 放棄自己的生活 - 認為「愛可以克服一切」

位置3：磨合期：最危險！

狀態： 熱戀過後，現實問題出現

典型問題：
- 生活習慣衝突
- 價值觀差異顯現
- 對未來的期望不同
- 溝通出現困難

位置3吉率僅6.7%：這是關係最容易結束的階段。

正確做法： - 認真對待衝突 - 學習有效溝通 - 適度妥協 - 評估核心不兼容性

錯誤做法： - 迴避問題 - 試圖改變對方 - 冷戰或爆發 - 在這個階段做重大決定（如結婚）

位置4：穩定期

狀態： 度過磨合期，開始形成默契

正確做法： - 建立共同的生活模式 - 尊重彼此的差異 - 持續維護關係 - 不要把對方視為理所當然

位置5：成熟期

狀態： 深厚的信任和支持

這是關係的「黃金期」：凶率0%。

正確做法： - 珍惜這個狀態 - 共同面對挑戰 - 繼續成長 - 為對方創造空間

位置6：轉變期

狀態： 關係需要更新

兩種可能： 1. 更新：關係進入新階段（如有孩子、退休後） 2. 結束：認識到關係無法繼續

正確做法： - 誠實面對 - 如果要結束，有尊嚴地結束 - 如果要更新，共同規劃

家庭關係中的位置

父母與子女

傳統結構：
父母 = 上卦（主導、提供）
子女 = 下卦（接受、學習）

但這個結構會隨時間變化：
- 子女長大後，結構趨向平等
- 父母年老後，結構可能反轉

關鍵智慧：認識到結構會變化

很多家庭衝突來自於： - 父母不接受子女已經長大 - 子女不理解父母的擔心 - 雙方還在用舊的結構互動

兄弟姐妹

理想結構：
- XOR值低（差異小）
- 各自有自己的「位置」
- 不爭搶同一個位置

衝突往往來自： - 都想當「老大」（都想佔位置5） - 覺得父母偏心 - 在同一領域競爭

職場人際關係

與上級的關係

你 = 下卦
上級 = 上卦

最佳組合：
- 你是艮（穩固、有底線）
- 上級是坤（支持、不壓制）
= 謙卦結構

較差組合：
- 你是兌（討好）
- 上級是乾（強勢）
= 權力不平衡

與上級相處的原則：

位置1（你是新人）：多觀察，少評判
位置2（你在成長）：展現能力，不搶風頭
位置3（你在轉折）：謹慎表達不滿
位置4-5（你是骨幹）：可以更直接，但尊重位階

與同事的關係

同事關係的挑戰：
- 既合作又競爭
- 沒有明確的位階
- 利益可能衝突

建議： - 明確各自的「領域」 - 在合作中互補 - 競爭時公平競爭 - 避免「兌在上」（表面和諧實則勾心鬥角）

與下屬的關係

你 = 上卦
下屬 = 下卦

最佳組合：
- 你是坤（支持、包容）
- 下屬有各種類型
= 讓下屬發揮

較差組合：
- 你是乾過度（控制、強勢）
- 下屬是坤過度（依賴、無主見）
= 團隊沒有活力

關係問題的「卦象診斷」

案例1：婚姻中的權力不平衡

情況： - 老公總是做決定 - 老婆感到沒有話語權 - 經常爭吵

卦象分析：

老婆（下卦）：坤（順從）→ 但感到壓抑
老公（上卦）：乾（主導）→ 認為這是負責任

結構 = 天地否（不通）

問題：「不交」：各自說各自的，不真正溝通

建議： - 老婆需要表達自己（從坤走向艮：有底線） - 老公需要傾聽（從乾走向坤：柔和一些） - 目標：從「否」（不通）轉向「泰」（通暢）

案例2：朋友關係漸行漸遠

情況： - 以前很好的朋友 - 最近聯繫越來越少 - 見面也沒話說

卦象分析：

你們的發展：
- 以前：位置2-3（熱絡期）
- 現在：位置6（轉變期）

可能原因：
- 生活環境變化（上卦不同了）
- 價值觀分歧（下卦不同了）
- XOR增大（差異變大）

建議： - 接受關係有生命週期 - 不必強求回到過去 - 可以讓關係轉型（從密友到普通朋友） - 有些關係「放手」是最好的

改善關係的「卦位調整」

調整你的「下卦」

如果你總是「兌」（討好）：
→ 嘗試變得更「艮」（有底線）

如果你總是「乾」（強勢）：

謙卦的結構

謙卦的結構：

     ━ ━  ┐
     ━ ━     上卦：坤（地）
     ━ ━  ┘
     ━━━  ┐
     ━ ━     下卦：艮（山）
     ━ ━  ┘

坤上艮下 = 地山謙

意象：高山藏於地下

一座高山本該高聳入雲，但它選擇藏在地下，不顯露自己。

這就是「謙」的本質：有實力，但不張揚。

謙卦原文

以下是謙卦的完整原文，供讀者對照參考：

卦辭

謙：亨，君子有終。

謙虛：亨通，君子會有好結局。

彖傳

彖曰：謙，亨。天道下濟而光明，地道卑而上行。天道虧盈而益謙，地道變盈而流謙，鬼神害盈而福謙，人道惡盈而好謙。謙尊而光，卑而不可踰，君子之終也。

解讀：天的法則是向下施恩而顯光明，地的法則是處於卑下而向上運行。天減損盈滿而增益謙虛，地改變盈滿而流向謙卑，鬼神禍害盈滿而賜福謙虛，人性厭惡盈滿而喜好謙虛。謙虛的人地位尊貴而有光彩，地位卑下卻不可逾越，這就是君子的結局。

象傳

象曰：地中有山，謙。君子以裒多益寡，稱物平施。

解讀：地下藏著山，這就是謙的象徵。君子因此減損多餘的，增益不足的，衡量事物而平均施予。

爻辭與小象

初六： > 謙謙君子，用涉大川，吉。 > > 象曰：謙謙君子，卑以自牧也。 > （謙虛再謙虛的君子，是以卑下的態度來修養自己。）

六二： > 鳴謙，貞吉。 > > 象曰：鳴謙貞吉，中心得也。 > （謙虛的名聲傳出，守正則吉，是因為內心真正領悟了謙的道理。）

九三： > 勞謙，君子有終，吉。 > > 象曰：勞謙君子，萬民服也。 > （有功勞而謙虛的君子，萬民都會信服。）

六四： > 無不利，撝謙。 > > 象曰：無不利撝謙，不違則也。 > （沒有不利的，發揚謙虛，是因為不違背準則。）

六五： > 不富以其鄰，利用侵伐，無不利。 > > 象曰：利用侵伐，征不服也。 > （適合採取行動，是為了征服不順從的。）

上六： > 鳴謙，利用行師，征邑國。 > > 象曰：鳴謙，志未得也。可用行師，征邑國也。 > （謙虛的美名傳出，但心願未完全達成。可以出兵，征伐城邑。）

最佳上卦（坤）+ 最佳下卦（艮）= 最佳整體（謙卦）

XOR值

艮 = 1 (001)

XOR = 1 是最小的非零差異：
內外有區別，但區別最小。

第二十四章：卦德交互：為什麼有些組合特別好或特別差

忘掉五行，回到卦德

如果你學過傳統易經，可能聽過「五行生剋」： - 金生水、水生木、木生火… - 金剋木、木剋土、土剋水…

聽起來很有道理，但數據告訴我們：五行對吉凶預測沒什麼用。

我們測試了五行關係，準確率只提升了0.3%左右，幾乎可以忽略。

那古人用的是什麼邏輯？

答案是：卦德。

八卦的「德」是什麼？

「德」在這裡不是道德的意思，而是「本質特性」或「性格」。

每個八卦都有一個核心特性：

卦      德  白話            想像畫面
────────────────────────────────────────────────────────
**乾**  健  一直衝、不停    天空永遠在動，不會停下來
**坤**  順  穩穩撐住、配合  大地承載萬物，穩定可靠
**震**  動  突然爆發        打雷，嚇人一跳
**巽**  入  慢慢滲透        風吹進縫隙，無孔不入
**坎**  陷  危險、陷阱      水坑，掉下去會出事
**離**  麗  依附、靠著      火需要東西燒，要靠著才能活
**艮**  止  停下來、擋住    山擋在路上，走不過去
**兌**  說  開心、交流      朋友聊天，心情好

這些特性來自《說卦傳》，是周朝時期就有的描述。

卦德怎麼影響吉凶？

當兩個卦組合在一起的時候，它們的「德」會互相作用。

好的組合：互補或加成

坤（穩定的地基）+ 震（向上的力量）
= 有穩定基礎，又有向上發展的動力
= 特別好！

統計數據：吉率 +0.83（遠高於單獨相加的 +0.26）

這就像蓋房子：地基穩（坤），然後往上蓋（震），當然成功率高。

坤（穩定）+ 巽（滲透）
= 有基礎，慢慢深入發展
= 也特別好！

統計數據：吉率 +0.83

這像是做生意：先站穩腳跟（坤），然後慢慢打入市場（巽）。

壞的組合：衝突或卡死

艮（停住）+ 艮（停住）
= 上面停，下面也停
= 動彈不得！

統計數據：吉率 -0.50（遠低於單獨相加的 +0.02）

這像是兩個人互相等對方先動，結果誰都不動，事情卡住了。

離（想靠穩）+ 震（一直晃）
= 想要依靠的東西一直在動
= 站不住！

統計數據：吉率 -0.50

這像是扶著一個搖晃的柱子，越扶越危險。

為什麼不是加法？

直覺上，我們會想：

坤的效應 = +0.375（好）
震的效應 = +0.167（還行）
坤+震 = +0.375 + 0.167 = +0.54？

但實際上：坤+震 = +0.83

多出來的 +0.29 是「交互效應」：兩個卦放在一起產生的額外作用。

這就像化學反應： - 氫氣（H）本身沒什麼 - 氧氣（O）本身也沒什麼 - 但 H + O = 水，完全不同的東西

卦德也是這樣。不是1+1=2，而是1+1可能等於3，也可能等於0。

用卦德理解每一卦

現在我們可以用卦德來「讀懂」64卦：

例子一：泰卦（坤上乾下）

下卦：乾（健，一直衝）
上卦：坤（順，穩穩撐住）

意思：下面有動力往上衝，上面穩穩地承接
結果：通達、順利

白話：做事有衝勁，又有人支持你，當然順利！

例子二：否卦（乾上坤下）

下卦：坤（順，穩穩的）
上卦：乾（健，一直衝）

意思：下面想穩，上面一直衝，方向相反
結果：閉塞、不通

白話：地基想穩，上面卻一直往天上衝，兩邊沒交集！

例子三：蹇卦（坎上艮下）

下卦：艮（止，停住）
上卦：坎（陷，危險）

意思：前面有危險，自己又停住不動
結果：困難重重

白話：看到危險，嚇到腿軟走不動，這下慘了！

例子四：渙卦（巽上坎下）

下卦：坎（陷，危險）
上卦：巽（入，滲透）

意思：有危險，但可以慢慢滲透過去
結果：散開、流通

白話：雖然前面有坑，但風可以吹過去，找到出路！

為什麼卦德比五行好用？

1. 更直覺

五行：
「火生土，土生金，金生水...」
→ 為什麼火生土？需要背誦，不直覺

卦德：
「坤（穩定）配震（向上）= 好」
→ 蓋房子的邏輯，不用背就懂

2. 更準確

我們測試了五行生剋關係，對準確率幾乎沒幫助（+0.3%）。 但卦德交互效應可以解釋那些「異常」的組合。

3. 更原始

卦德來自《說卦傳》，是周朝時期的文獻。 五行大量應用到易經是漢代以後的事。

卦德才是易經的原始設計邏輯。

實際應用

當你遇到一個卦的時候：

第一步：拆解上下卦

例如：山雷頤
下卦：震（雷，動）
上卦：艮（山，止）

第二步：想像畫面

下面在動（震），上面擋住（艮）
→ 能量往上衝，但被壓住
→ 要小心，有壓力

第三步：查交互效應

震下+艮上 的統計吉率是多少？
→ 查表可知

這比背誦64卦的意思有效多了。理解卦德，就能理解任何組合。

Appendix A2: Research Methodology

研究概述

本書的數據發現基於對易經的系統性計算分析。

我們把易經視為一個「黑盒子」：輸入結構特徵，輸出吉凶結果：然後用統計方法尋找規律。

數據來源

分析樣本

  • 完整卦數量：64卦
  • 完整爻數量：64 × 6 = 384個爻
  • 所有統計均基於完整384爻數據

吉凶判定標準

爻辭的吉凶根據傳統易學判定，分為三類：

吉（+1）：爻辭中有「吉」「利」「亨」等正面結果
凶（-1）：爻辭中有「凶」「厲」「咎」等負面結果
平（0）：爻辭中性，或有條件性結果

分析方法

使用的27種技術

統計分析：
- 描述性統計
- 交叉表分析
- 卡方檢驗
- 相關分析
- 回歸分析

機器學習：
- 決策樹

- 神經網絡

模式識別：
- 聚類分析
- 關聯規則
- 序列模式
- 頻繁項集

信息論：
- 熵分析

其他：

- Gray碼分析
- 網格位置分析
- 對稱性分析

特徵工程

我們從每個爻提取以下結構特徵：

位置特徵：
- 爻位（1-6）
- 是否得中（位置2或5）
- 是否在上卦/下卦
- 距離上下卦邊界的位置

陰陽特徵：
- 爻的陰陽（0或1）
- 是否「當位」（陽爻在奇數位，陰爻在偶數位）

卦結構特徵：
- 上卦類型（0-7）
- 下卦類型（0-7）
- 上下卦XOR值
- 是否純卦（上下卦相同）
- Gray碼距離

組合特徵：
- 位置 × 陰陽
- 位置 × XOR
- 上卦 × 下卦

主要發現的統計檢驗

發現1：位置5從不凶

數據：64個第五爻中，0個被判定為凶
統計檢驗：二項檢驗
零假設：凶率 = 全體平均凶率（約25%）
結果：p < 0.001，拒絕零假設
結論：位置5凶率顯著低於平均，統計上確認「從不凶」

發現2：位置3最危險

數據：位置3的吉率僅6.7%（6/90）
對比：全體平均吉率約28%
統計檢驗：卡方檢驗
結果：χ² = 15.3, p < 0.001
結論：位置3吉率顯著低於平均

發現3：吉率最高的卦

數據：臨卦、萃卦、謙卦均為83.3%吉率（5吉0凶）
注：沒有任何卦是100%六爻皆吉
分析：
- 離作為上卦的吉率：56.2%（最高）
- 坤作為下卦的吉率：54.2%（最高）
結論：上下卦組合影響整體吉凶

發現4：適度差異法則

數據：按XOR差異維度分組的吉率
- 0維度差異（XOR=0）：31.2%

- 3維度差異（XOR=7）：41.7%

Q: 什麼是「當位」？ A: 陽爻在奇數位（1、3、5），陰爻在偶數位（2、4、6），叫當位。

Q: 「用九」「用六」是什麼？ A: 乾卦和坤卦獨有的附加爻辭，代表所有爻都動時的情況。

基本概念

每個卦有6個爻（線）。變一個爻，就會變成另一個卦。

例：乾卦變第4爻 → 履卦

乾卦          變第4爻        履卦
━━━ 第6爻     ━━━           ━━━
━━━ 第5爻     ━━━           ━━━
━━━ 第4爻  →  ━ ━ (陽變陰)   ━ ━
━━━ 第3爻     ━━━           ━━━
━━━ 第2爻     ━━━           ━━━
━━━ 第1爻     ━━━           ━━━

表格怎麼看

當你看到「4→履」，意思是：

寫法 意思
4→履 變第4爻，會變成履卦
6→臨 變第6爻，會變成臨卦
3→謙 變第3爻，會變成謙卦

實際應用：不是改卦象，是改行動

重要：你不是真的去「改變」卦象。卦象代表你的現狀。

「變爻」的意思是：改變那個位置代表的事情。

爻位代表的層面：
第1爻 = 起步、基礎、根本
第2爻 = 發展、穩定、內在
第3爻 = 轉折、決定、風險點
第4爻 = 適應、調整、外在開始
第5爻 = 主導、發揮、核心位置
第6爻 = 終結、轉型、結束階段

完整範例

情境：你占到「師卦」，查表發現是🔴排斥子，建議「走」，最佳變爻「6→臨」。

解讀： 1. 師卦 = 你目前在帶領團隊打仗的狀態，吉率只有17% 2. 建議「走」= 這個狀態不適合久留 3. 「6→臨」= 改變第6爻的層面（終結、轉型） 4. 臨卦 = 主動面對、進入的狀態，吉率83%

行動建議： - 第6爻代表「結束階段」 - 所以你應該思考：如何結束目前的「打仗」狀態，轉向「主動面對」的狀態 - 具體可能是：結束對抗、結束消耗戰、轉向合作或主動出擊

簡單記憶

看到「X→某卦」：
1. X = 要改變的層面（1=根本, 2=發展, 3=決定, 4=調整, 5=主導, 6=轉型）
2. 某卦 = 改變後會進入的狀態
3. 行動 = 在那個層面做出改變
