# 爻辭完整解釋：第1-8卦

## 解讀方法

每爻解釋包含：
1. **爻辭**：原文
2. **象傳**：傳統解釋（為什麼這樣說）
3. **爻位分析**：位置、得位、得中、相應、承乘
4. **白話解釋**：現代理解

---

## 第1卦 乾為天 ䷀

**卦義**：純陽剛健，創造力的完整週期

### 初九：潛龍勿用

| 項目 | 內容 |
|------|------|
| **爻辭** | 潛龍，勿用 |
| **象傳** | 陽在下也 |
| **文言傳** | 龍德而隱者也，不易乎世，不成乎名，遯世無悶 |

**爻位分析**：
- 位置：初爻（萌芽期）
- 陰陽：九（陽爻）
- 得位：✓（陽在陽位）
- 得中：✗
- 相應：與九四（同性無應）

**為什麼「勿用」**：
1. 象傳說「陽在下」→ 陽氣在最底層，力量尚未顯現
2. 初爻代表事物的開始階段，不宜行動
3. 「潛」= 潛伏蓄力，時機未到

**白話**：能力在培養中，時機未成熟，宜韜光養晦、積蓄力量，不宜急於表現。

- 位置：五爻（君位）
- 陰陽：六（陰爻）
- 得中：✓
- 相應：與九二有應 ✓

**為什麼「長子帥師」vs「弟子輿尸」**：
1. 六五是君位，但陰爻 → 需要用人
2. 「長子」= 九二（陽爻、有能力）→ 正確人選
3. 「弟子」= 其他陰爻 → 用人不當
4. 「使不當」= 任命不恰當則凶

**白話**：有敵入侵，宜討伐。用長子（能幹者）統帥軍隊可行；用弟子（不適任者）則會載屍而歸。用人是否得當決定成敗。
